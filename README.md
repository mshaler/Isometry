# Isometry

A polymorphic data visualization platform implementing PAFV + LATCH + GRAPH.

## Quick Start

```bash
npm install
npm run dev
```

- **Visualization**: D3.js
- **UI**: React + Tailwind
- **Drag-Drop**: react-dnd
