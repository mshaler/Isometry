Apple pixels

| **Model** | **Screen Size** | **Resolution** | **PPI** | **Brightness (nits)** |

|---|---|---|---|---|

| **MacBook Air** |  |  |  |  |

| MacBook Air 13" (M2/M3) | 13.6" | 2560 × 1664 | 224 | 500 |

| MacBook Air 15" (M2/M3) | 15.3" | 2880 × 1864 | 224 | 500 |

| **MacBook Pro** |  |  |  |  |

| MacBook Pro 14" (M3/M3 Pro/M3 Max) | 14.2" | 3024 × 1964 | 254 | 1,000 (1,600 peak HDR) |

| MacBook Pro 16" (M3 Pro/M3 Max) | 16.2" | 3456 × 2234 | 254 | 1,000 (1,600 peak HDR) |

| **External Displays** |  |  |  |  |

| Studio Display | 27" | 5120 × 2880 | 218 | 600 |

| Pro Display XDR | 32" | 6016 × 3384 | 218 | 1,000 (1,600 peak HDR) |

**Key Notes:**

MacBook Pro models support HDR content with higher peak brightness

Pro Display XDR offers professional-grade HDR performance matching MacBook Pro peak brightness

Studio Display provides excellent sharpness but standard brightness levels

Both external displays have the same PPI despite different sizes, similar to the MacBook Air lineup
