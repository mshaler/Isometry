CardBoard next steps

 Audit mode	- Turns off other formatting (fun police)

| Criteria | Text Formatting | Background formatting | Border formatting |

|---|---|---|---|

| Source data | Dark Gray | Light gray | Dark gray |

| Calculated values (toggles formulas and values) | Blue | Light blue | Blue |

| Enrichment/augmentation (such as standard ontology) | Green | Light green | Green |

| CRUD | Red | Light Red | Red |

 Anomoly detection	- Data cell collision

* Stray formula cells
* Ugly formatting

 Version control	- Every change is captured, like Google Sheets, with major and minor versions

 Need to refactor and run proper code review

 Need to fix resizing, state management, drag/drop, pin to left corner

 Data detectors	- Like Apple Mail (addresses, emails, URL cards)

* Callouts
* Tasks

Links (eventually automated backlinks)
