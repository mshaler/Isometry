**Step 2 — Create PanelManager.ts** with the following design (per D-01, D-02, D-03):

```typescript
import type { PanelRegistry } from './PanelRegistry';
import type { SlotConfig, CouplingGroup } from './PanelTypes';

export class PanelManager {
  private _registry: PanelRegistry;
  private _slots: Map<string, SlotConfig>;       // panel id -> slot config
  private _mounted: Set<string>;                  // panel ids that have been mounted at least once
  private _visible: Set<string>;                  // panel ids currently visible (display !== 'none')
  private _groups: Map<string, CouplingGroup>;    // group name -> coupling group
  private _syncSlots: () => void;                 // callback to sync slot container visibility

  constructor(config: {
    registry: PanelRegistry;
    slots: SlotConfig[];
    groups?: CouplingGroup[];
    syncSlots: () => void;
  });

  show(id: string): void;    // enable+mount on first call, toggle display on subsequent
  hide(id: string): void;    // set display='none', do NOT disable/destroy
  toggle(id: string): void;  // flip visible state
  isVisible(id: string): boolean;
  showGroup(name: string): void;   // show all panels in named group
  hideGroup(name: string): void;   // hide all panels in named group
  isGroupVisible(name: string): boolean;  // true if ANY panel in group is visible
}
```

Implementation details:
- `show(id)`: If not in `_mounted`, call `_registry.enable(id)` which triggers factory. Get instance via `_registry.getInstance(id)`. Call `instance.mount(container)` where container comes from `_slots.get(id)`. Add to `_mounted`. Set `container.style.display = ''`. Add to `_visible`. Call `_syncSlots()`.
- `hide(id)`: If not in `_visible`, return. Set `container.style.display = 'none'`. Remove from `_visible`. Call `_syncSlots()`. Do NOT call `_registry.disable()` — panels stay mounted (D-03).
- `toggle(id)`: Call `isVisible(id) ? hide(id) : show(id)`.
- `showGroup(name)`: Get group by name, call `show()` for each panelId.
- `hideGroup(name)`: Get group by name, call `hide()` for each panelId.
- `isVisible(id)`: Return `_visible.has(id)`.
- `isGroupVisible(name)`: Return true if any panel in group is visible.
