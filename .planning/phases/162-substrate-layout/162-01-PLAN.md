Create `src/superwidget/SuperWidget.ts`:

**1. CSS import** at top:
```typescript
import '../styles/superwidget.css';
```
This is the only style injection mechanism — no link tags (per SLAT-06 / D-01).

**2. Class definition:**
```typescript
export class SuperWidget {
  private _root: HTMLElement;
  private _headerEl: HTMLElement;
  private _canvasEl: HTMLElement;
  private _statusEl: HTMLElement;
  private _tabsEl: HTMLElement;
  private _mounted: boolean = false;
```

**3. Constructor** — builds all DOM in memory (does NOT append to document):
- Create root div: `document.createElement('div')`, set `dataset.component = 'superwidget'`
- Create header slot: `document.createElement('div')`, set `dataset.slot = 'header'`, set `textContent = 'Zone'` (placeholder per UI-SPEC)
- Create tabs slot: `document.createElement('div')`, set `dataset.slot = 'tabs'`
  - Create 3 placeholder tab buttons (`document.createElement('button')`), each with `dataset.tabRole = 'tab'` and textContent "Tab 1"/"Tab 2"/"Tab 3". First tab gets `dataset.tabActive = 'true'`.
  - Create config gear button: `document.createElement('button')`, `dataset.tabRole = 'config'`, `textContent = '\u2699'` (gear unicode), `setAttribute('aria-label', 'Configure')`
  - Append all buttons to tabs slot
- Create canvas slot: `document.createElement('div')`, set `dataset.slot = 'canvas'`
- Create status slot: `document.createElement('div')`, set `dataset.slot = 'status'`
- Append to root in grid order: header, tabs, canvas, status (per D-07)

**4. mount(container: HTMLElement): void**
- Guard: if already mounted, return (idempotent)
- `container.appendChild(this._root)`
- Set `this._mounted = true`

**5. destroy(): void**
- Guard: if not mounted, return
- `this._root.remove()`
- Set `this._mounted = false`
- Null all slot references (set to null! or use a destroyed flag)

**6. Public read-only getters** (per D-05):
```typescript
get headerEl(): HTMLElement { return this._headerEl; }
get canvasEl(): HTMLElement { return this._canvasEl; }
get statusEl(): HTMLElement { return this._statusEl; }
get tabsEl(): HTMLElement { return this._tabsEl; }
get rootEl(): HTMLElement { return this._root; }
get mounted(): boolean { return this._mounted; }
```

Do NOT add any event handlers (per D-06). No state management. This is pure DOM skeleton + lifecycle.
