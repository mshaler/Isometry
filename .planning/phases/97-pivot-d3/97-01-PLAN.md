## Conversion Map

| React Pattern | D3/Vanilla Equivalent |
|--------------|----------------------|
| `useState` | Class field + `_notify()` re-render |
| `useMemo` | Recompute in `_render()` only when deps change |
| `useRef` | Direct `this._el` DOM reference |
| `useEffect` (resize) | `pointerdown/pointermove/pointerup` on document |
| `DndProvider + HTML5Backend` | Removed entirely |
| `useDrag` | `pointerdown` on chip ã ghost + `setPointerCapture` |
| `useDrop` | `pointerup` hit-test via `elementsFromPoint` |
| JSX elements | `d3.select().append()` chains |
| Tailwind classes | CSS custom properties + semantic classes |
| `React.UIEvent<HTMLDivElement>` scroll | Native `scroll` event listener |

## Anti-Patterns to Avoid
- No `dragstart`/`dragover`/`drop` events (HTML5 DnD) ã pointer events only
- No framework dependencies (React, etc.)
- No Tailwind ã use CSS custom properties matching project convention
- No inline styles except grid-row/grid-column positioning (matching D-pattern)
