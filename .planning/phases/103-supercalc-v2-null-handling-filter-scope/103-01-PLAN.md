    4. Implement logic:
    - If fn === 'NONE': return { value: null }
    - If nullMode === 'strict' and values.some(v => v === null): return { value: null, warning: 'incomplete-data' }
    - If fn === 'COUNT':
      - If countMode === 'all': return { value: values.length }
      - Else (column): return { value: values.filter(v => v !== null).length } (count original non-nulls, regardless of nullMode)
    - If values.length === 0: return { value: null }
    - For nullMode === 'zero': create substituted = values.map(v => v ?? 0), use substituted for SUM/AVG/MIN/MAX
    - For nullMode === 'exclude': use existing logic (filter non-nulls for AVG/MIN/MAX, treat null as 0 for SUM)
    - SUM: reduce with + over working array, return { value: sum }
    - AVG with 'zero': sum / values.length (total rows). AVG with 'exclude': sum / nonNulls.length
    - MIN/MAX: Math.min/max over working array. With 'zero', 0 is a candidate.
    - If nonNulls empty (exclude mode): return { value: null }
