**Change 3: Add _applyActiveRowHighlight() method and call it after refresh**
Add a private method to CatalogSuperGrid:

```typescript
private _applyActiveRowHighlight(): void {
  if (!this._container) return;
  const activeKey = this._bridgeAdapter.activeRowKey;
  // Remove from all cells first
  const allCells = this._container.querySelectorAll('[data-row-key]');
  for (const cell of allCells) {
    cell.classList.remove('dexp-catalog-row--active');
  }
  // Apply to active row cells
  if (activeKey) {
    const activeCells = this._container.querySelectorAll(
      `[data-row-key="${CSS.escape(String(activeKey))}"]`
    );
    for (const cell of activeCells) {
      cell.classList.add('dexp-catalog-row--active');
    }
  }
}
```

**Change 4: Call _applyActiveRowHighlight after every SuperGrid render**
In the `mount()` method, after `this._superGrid.mount(container);` (line 236), use a MutationObserver on the container to detect when SuperGrid finishes rendering cells (child list changes), then apply the highlight:

const observer = new MutationObserver(() => {

observer.observe(container, { childList: true, subtree: true });
