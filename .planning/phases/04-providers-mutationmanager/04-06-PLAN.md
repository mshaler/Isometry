3. `deleteCardMutation(card: Card): Mutation`
   - Forward: `DELETE FROM cards WHERE id = ?`
   - Inverse: `INSERT INTO cards (id, card_type, name, content, ...) VALUES (?, ?, ?, ?, ...)`
   - Must include ALL 25 columns in the INSERT
   - Handle `tags`: serialize array to JSON string
   - Handle `is_collective`: convert boolean to 0/1 for SQLite

4. `createConnectionMutation(input: ConnectionInput): Mutation`

   - Forward: INSERT INTO connections

Create `src/mutations/MutationManager.ts`:

```typescript
const MAX_HISTORY = 100;

class MutationManager {
  private history: Mutation[] = [];
  private redoStack: Mutation[] = [];
  private dirty = false;
  private pendingNotify = false;
  private subscribers = new Set<() => void>();

  constructor(private bridge: WorkerBridge) {}

  async execute(mutation: Mutation): Promise<void>;
  async undo(): Promise<boolean>;
  async redo(): Promise<boolean>;
  canUndo(): boolean;
  canRedo(): boolean;
  getHistory(): readonly Mutation[];
  isDirty(): boolean;
  clearDirty(): void;
  subscribe(cb: () => void): () => void;
  private scheduleNotify(): void;
}
```

Key implementation:
- `execute(mutation)`: For each cmd in mutation.forward: await bridge.send('db:exec', { sql: cmd.sql, params: cmd.params }). Push to history. If history.length > MAX_HISTORY, shift oldest (console.warn on first overflow). Clear redoStack. Set dirty = true. scheduleNotify().
- `undo()`: Pop from history. For each cmd in [...mutation.inverse].reverse(): await bridge.send('db:exec', ...). Push to redoStack. Set dirty. scheduleNotify().
- `redo()`: Pop from redoStack. For each cmd in mutation.forward: await bridge.send('db:exec', ...). Push to history. Set dirty. scheduleNotify().
- `scheduleNotify()`: `if (pendingNotify) return; pendingNotify = true; requestAnimationFrame(() => { pendingNotify = false; subscribers.forEach(cb => cb()); });`
