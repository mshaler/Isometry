       - Calculate the 7 dates of the week containing the current date
       - Set gridContainer to 7-column grid
       - Create 7 day cells with dataset.date

       _buildDayView(year, month, day):
       - Single column, single day cell

       _buildQuarterView(year, quarter):
       - Create 3 mini-month containers (div.mini-month) in a 3-column grid
       - Each mini-month has a label + simplified month grid (small cells, no card chips)

       _buildYearView(year):
       - Create 12 mini-month containers in a 4-column by 3-row grid

         - If remaining > 0: append span.overflow-label with "+N more" text

       _navigate(direction):
       - Adjust currentMonth/currentYear by direction based on current granularity
       - Call _buildStructure() and re-bind with last rendered cards

       destroy():
