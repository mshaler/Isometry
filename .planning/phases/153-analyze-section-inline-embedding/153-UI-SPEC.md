### Bottom Slot Child Div Spec

```
  style.display = 'none'   (hidden until first toggle-on)
  width: 100%
```

```css
/* Bottom-slot child containers — one per explorer (Phase 153) */
```

### Slot Visibility Behavior

`syncBottomSlotVisibility()` mirrors `syncTopSlotVisibility()`:
- `.workbench-slot-bottom` display = `''` (visible) when at least one child has `display !== 'none'`
- `.workbench-slot-bottom` display = `'none'` when all children have `display === 'none'`

Initial state: both children hidden → `.workbench-slot-bottom` hidden (display: none).

### Stacking Order (D-05)

Fixed DOM order. Filters always renders above Formulas. When both are visible:
- Filters occupies its natural height (auto, driven by LatchExplorers content)
- Formulas renders below with `border-top: 1px solid var(--border-subtle)` via `.slot-bottom__formulas-explorer` CSS rule

## Interaction Contract

### Toggle Behavior (ANLZ-01, ANLZ-02, ANLZ-04, ANLZ-05)

| User Action | State Change | Visual Feedback |
|-------------|-------------|-----------------|
| Click "Filters" dock item (off → on) | Lazy-mount LatchExplorers into `.slot-bottom__latch-filters`, set display = '' | Dock item aria-pressed="true" (accent background fill), bottom slot appears |
| Click "Filters" dock item (on → off) | Set `.slot-bottom__latch-filters` display = 'none', syncBottomSlotVisibility() | Dock item aria-pressed="false" (no accent), slot hides if Formulas also off |
| Click "Formulas" dock item (off → on) | Lazy-mount FormulasPanelStub into `.slot-bottom__formulas-explorer`, set display = '' | Dock item aria-pressed="true", bottom slot appears (or stays visible) |
| Click "Formulas" dock item (on → off) | Set `.slot-bottom__formulas-explorer` display = 'none', syncBottomSlotVisibility() | Dock item aria-pressed="false", slot hides if Filters also off |

### Cross-View Persistence (ANLZ-03)

Mechanism: DOM sibling survival (D-04). The `.workbench-slot-bottom` element is a flex sibling of `.workbench-view-content`. View switches replace content inside `.workbench-view-content` only. The bottom slot is unaffected.

No guard logic required. No state variable tracking across view switches. The `display` property on child divs is the persistent state.

### Dock Item Pressed State (D-08)

`dockNav.setItemPressed('analyze:filter', bool)` sets `aria-pressed` attribute on the Filters dock item button.
`dockNav.setItemPressed('analyze:formula', bool)` sets `aria-pressed` attribute on the Formulas dock item button.

Matches Phase 152 `integrate:catalog` pattern.

Pressed state visual (from `src/styles/dock-nav.css`):
- `aria-pressed="true"` → `.dock-nav__item--active` styles apply (accent background fill, --text-primary color)
- `aria-pressed="false"` → default `.dock-nav__item` styles (transparent background, --text-secondary)

### Lazy Mount Pattern

Filters:
```
latchFiltersMounted: boolean = false

showLatchFilters():
  if (!latchFiltersMounted):
    latchExplorers = new LatchExplorers({ filter, bridge, coordinator, schema })
    latchExplorers.mount(latchFiltersChildEl)
    latchFiltersMounted = true
  latchFiltersChildEl.style.display = ''
  syncBottomSlotVisibility()
  dockNav.setItemPressed('analyze:filter', true)

hideLatchFilters():
  latchFiltersChildEl.style.display = 'none'
  syncBottomSlotVisibility()
  dockNav.setItemPressed('analyze:filter', false)
```

Formulas:
```
formulasMounted: boolean = false

showFormulasExplorer():
  if (!formulasMounted):
    formulasPanelFactory().mount(formulasChildEl)
    formulasMounted = true
  formulasChildEl.style.display = ''
  syncBottomSlotVisibility()
  dockNav.setItemPressed('analyze:formula', true)

hideFormulasExplorer():
  formulasChildEl.style.display = 'none'
  syncBottomSlotVisibility()
  dockNav.setItemPressed('analyze:formula', false)
```
