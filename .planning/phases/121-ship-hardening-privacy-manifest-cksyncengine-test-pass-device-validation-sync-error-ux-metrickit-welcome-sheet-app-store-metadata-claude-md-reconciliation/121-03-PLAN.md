1. **Create WelcomeSheet.swift** at `native/Isometry/Isometry/WelcomeSheet.swift`:

   Per UI-SPEC layout contract exactly:
   ```swift
   import SwiftUI

   struct WelcomeSheet: View {
       let onLoadSampleData: () -> Void
       let onStartEmpty: () -> Void

       var body: some View {
           VStack(spacing: 0) {
               Spacer()
                   .frame(height: 48)  // 2xl top inset

               // App icon -- 80x80pt
               Image("AppIcon")  // or use Image(nsImage:/uiImage: from bundle)
                   .resizable()
                   .frame(width: 80, height: 80)
                   .clipShape(RoundedRectangle(cornerRadius: 18))
                   .accessibilityLabel("Isometry app icon")

               Spacer()
                   .frame(height: 16)  // md gap

               // App name
               Text("Isometry")
                   .font(.title2)
                   .fontWeight(.semibold)
                   .accessibilityAddTraits(.isHeader)

               // Tagline
               Text("See everything, connected.")
                   .font(.headline)
                   .foregroundStyle(.secondary)

               Spacer()
                   .frame(height: 24)  // lg gap

               // Primary CTA
               Button {
                   onLoadSampleData()
               } label: {
                   Text("Load Sample Data")
                       .frame(maxWidth: .infinity)
               }
               .buttonStyle(.borderedProminent)
               .controlSize(.large)

               Spacer()
                   .frame(height: 12)  // sm gap between buttons

               // Secondary CTA
               Button {
                   onStartEmpty()
               } label: {
                   Text("Start Empty")
                       .frame(maxWidth: .infinity)
               }
               .buttonStyle(.bordered)
               .controlSize(.large)

               Spacer()
                   .frame(height: 48)  // 2xl bottom inset
           }
           .padding(.horizontal, 32)
           .interactiveDismissDisabled(true)  // No swipe-to-dismiss on first launch
       }
   }
   ```

   Notes on app icon: If the asset catalog icon is not directly accessible via `Image("AppIcon")`, use an alternative approach:
   - On iOS: `Image(uiImage: UIImage(named: "AppIcon") ?? UIImage())` -- but AppIcon is usually not directly loadable from asset catalog. Instead, add a separate `AppIconDisplay` image to the asset catalog at 80x80pt, OR use SF Symbol `app.fill` as fallback.
   - Simplest approach: Add a 80x80pt `WelcomeIcon` image set to the asset catalog, or use `Image(systemName: "square.grid.3x1.folder.badge.plus")` as an interim icon.
   - Check the project's Assets.xcassets for an existing AppIcon set and determine the best approach.

2. **Wire WelcomeSheet into ContentView.swift**:
   - Add `@AppStorage("hasSeenWelcome") private var hasSeenWelcome = false`
   - Add `@State private var showingWelcome = false`
   - In `.onAppear` block, add: `if !hasSeenWelcome { showingWelcome = true }`
   - Add `.sheet(isPresented: $showingWelcome)` modifier:
     ```swift
     .sheet(isPresented: $showingWelcome) {
         WelcomeSheet(
             onLoadSampleData: {
                 hasSeenWelcome = true
                 showingWelcome = false
                 // Trigger SampleDataManager via JS bridge
                 Task {
                     try? await bridgeManager.webView?.evaluateJavaScript(
                         "window.__isometry?.sampleDataManager?.generate()"
                     )
                 }
             },
             onStartEmpty: {
                 hasSeenWelcome = true
                 showingWelcome = false
             }
         )
     }
     ```
   - Important: The welcome sheet should present AFTER the webView is ready (native:ready received). Check `bridgeManager.isJSReady` (or equivalent flag) before showing. If JS is not ready, delay showing the sheet until it is (use onChange of bridgeManager.isJSReady). This ensures "Load Sample Data" can actually call the JS bridge.
