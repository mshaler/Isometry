# TestFlight External Beta Rollout Checklist (Phase 13.3)

## 1) Preconditions
- [ ] Apple Developer program active and correct Team ID available
- [ ] App Store Connect API key available (optional but recommended)
- [ ] Distribution certificates installed in Keychain
- [ ] Latest production settings merged into release branch

## 2) App Store Connect Setup
- [ ] Open App Store Connect → My Apps → Isometry → TestFlight
- [ ] Confirm Export Compliance (encryption settings) complete
- [ ] Fill Beta App Review Info (contact + notes)
- [ ] Create External Testing group (e.g., "Isometry Beta")
- [ ] Add external testers (email invites)

## 3) Build & Upload (beta-only)
- [ ] Run beta-only pipeline:
  - `./scripts/build-and-upload-appstore.sh --beta-only`
- [ ] Verify build uploaded in App Store Connect → TestFlight → Builds
- [ ] Submit build for TestFlight review

## 4) Review & Distribution
- [ ] Wait for build status to reach "Ready to Test"
- [ ] Enable External Testing for the build
- [ ] Confirm tester invites delivered and accepted
- [ ] Verify install + launch on at least 2 devices

## 5) Feedback Collection
- [ ] Enable feedback capture (in-app feedback hookup + TestFlight feedback)
- [ ] Confirm beta feedback channels available to testers
- [ ] Monitor TestFlight crash reports and feedback notes

## 6) Verification Quick Checks
- [ ] Beta build launches without crash
- [ ] CloudKit sync initiates and completes
- [ ] Monitoring/analytics logs visible in beta session
- [ ] Feedback submission persists successfully

## Notes
- Use this checklist during the first external beta rollout.
- Record any deviations or delays in the Phase 13.3 quality report.
