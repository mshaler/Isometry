  <name>Task 1: Create adapter function with type mappings</name>
  <files>src/components/supergrid/adapters/headerTreeAdapter.ts</files>
  <action>
Create the adapter directory and file with:

1. Import types:
   - `HeaderTree`, `HeaderNode`, `FacetConfig` from `@/superstack/types/superstack`
   - `AxisConfig`, `AxisNode`, `LATCHAxisType` from `../types`

2. Export `headerTreeToAxisConfig(headerTree: HeaderTree): AxisConfig`:
   - Extract primary facet: `headerTree.facets[0]`
   - Map facet.axis to LATCHAxisType (both use 'L' | 'A' | 'T' | 'C' | 'H')
   - Create root AxisNode with label: 'Root', id: 'root'
   - Convert `headerTree.roots` to children via `convertHeaderNode`
   - Return AxisConfig: `{ type, facet: primaryFacet.sourceColumn, tree }`

3. Export `convertHeaderNode(node: HeaderNode): AxisNode`:
   - Map `node.label` to `axisNode.label`
   - Map `node.id` to `axisNode.id`
   - Map `node.span` to `axisNode.leafCount`
   - Set `expandable: node.children.length > 0 && !node.collapsed`
   - Set `expanded: !node.collapsed` (invert collapsed flag)
   - Recursively convert children if `node.children.length > 0`
   - Return undefined for children if no children (not empty array)

4. Handle edge cases:
   - Empty roots array: return tree with empty children array
   - Empty facets array: throw descriptive error (can't determine axis type)
   - Null/undefined headerTree: throw TypeError

Document the field mapping in JSDoc comments.
  </action>
