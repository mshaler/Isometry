  <name>Task 1: Create useGridDataCells hook</name>
  <files>src/hooks/useGridDataCells.ts</files>
  <action>
Create the hook with:

1. Import types:
   - `useMemo` from 'react'
   - `useDatabaseService` from '@/hooks/useDatabaseService'
   - `DataCell` from '@/components/supergrid/types'
   - `FacetConfig` from '@/superstack/types/superstack'

2. Export `computeNodePath(node: Record<string, unknown>, facets: FacetConfig[]): string[]`:
   - Map over facets
   - For each facet, extract value from node using `node[facet.sourceColumn]`
   - Convert value to string, use '(empty)' for null/undefined
   - Handle multi_select dataType by extracting first value if JSON array
   - Return string array of path values

3. Export `useGridDataCells(options: UseGridDataCellsOptions): DataCell[]`:
   - Options interface: `{ rowFacets: FacetConfig[], colFacets: FacetConfig[], whereClause?: string, parameters?: (string | number)[], nodeType?: string }`
   - Get database service via `useDatabaseService()`
   - Use `useMemo` with dependencies: `[db, rowFacets, colFacets, whereClause, parameters, nodeType]`
   - Early return empty array if `!db?.isReady()` or `rowFacets.length === 0` or `colFacets.length === 0`

4. SQL query construction:
   - Select columns: `n.id, n.name, n.folder, n.tags, n.status, n.priority, n.created_at, n.modified_at, n.node_type`
   - WHERE: `deleted_at IS NULL` plus optional whereClause and node_type filter
   - Use parameterized query with `db.exec(sql, params)`

5. Transform results to DataCell[]:
   - For each row, create node object from column values
   - Compute rowPath via `computeNodePath(node, rowFacets)`
   - Compute colPath via `computeNodePath(node, colFacets)`
   - Create DataCell: `{ rowPath, colPath, value: node.name as string, rawValue: node }`

6. Return DataCell array
