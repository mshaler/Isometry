    **ChartRenderer.ts** — Orchestrates chart lifecycle (mount, update, destroy).

    Constructor config: `{ bridge: WorkerBridge, filter: FilterProvider, alias: AliasProvider }`.

    Public API:
    - `mountCharts(previewEl: HTMLElement): Promise<void>` — Find all `.notebook-chart-card` divs in previewEl, parse their `data-chart-config` (base64-decoded), resolve field aliases, send `chart:query` to Worker, dispatch to type-specific renderer. On parse error or field validation failure, render inline error via `_renderError(div, message)`.
    - `startFilterSubscription(): () => void` — Subscribe to FilterProvider changes. On change, debounce 300ms, then re-query and update all active chart containers in-place. Returns unsubscribe function.
    - `destroyCharts(): void` — Remove all active chart SVGs and clear state.

    Internal state:
    - `_activeCharts: Map<string, { config: ChartConfig, container: HTMLDivElement }>` — Track mounted charts by chart ID for in-place updates.
    - `_queryGeneration: number` — Increment before each query batch, compare on response to discard stale results (race condition guard per RESEARCH.md Pitfall 6).
    - `_debounceTimer: ReturnType<typeof setTimeout> | null` — For filter change debounce.

    Reverse alias resolution helper:
    ```typescript
    private _resolveField(displayName: string): string | null
    ```
    Iterate `ALLOWED_AXIS_FIELDS`, check if `alias.getAlias(field) === displayName` — if match, return field. Also check if displayName itself is in the allowlist (user may use column names directly). Return null if unresolvable.

    **BarChart.ts** — `renderBarChart(container: HTMLDivElement, data: { label: string; value: number }[], config: ChartConfig): void`
    - Aspect ratio: 5:3 (height = width * 0.6)
    - Margins: top 20, right 20, bottom 40, left 50
    - d3.scaleBand for x-axis, d3.scaleLinear for y-axis
    - D3 data join with `.data(data, d => d.label)` key function (mandatory per D-003)
    - `.join(enter, update, exit)` with 300ms transitions on enter (grow from y=0) and update (morph) and exit (shrink to y=0 and remove)
    - Fill: config.color if specified, otherwise `d3.schemeTableau10[0]`
    - Hover tooltip: absolute-positioned `.notebook-chart-tooltip` div showing "label: value" on mouseover, hidden on mouseout
    - X-axis: `.ticks(5)` to prevent label overlap, rotated 30deg if labels would overlap
    - Y-axis: `d3.format('~s')` for SI-prefix formatting (1K, 2M)
    - Title: If config.title, render `<text>` element centered at top

    **PieChart.ts** — `renderPieChart(container: HTMLDivElement, data: { label: string; value: number }[], config: ChartConfig): void`
    - Square SVG: width = container width, height = width (1:1)
    - d3.pie().value(d => d.value).sort(null) — preserve data order
    - d3.arc() with innerRadius = radius * 0.5 (donut), outerRadius = radius - 10
    - D3 data join on arc paths with `.data(arcs, d => d.data.label)` key function
    - 300ms transitions on arc path changes (d3.interpolate on startAngle/endAngle)
    - Fill: d3.schemeTableau10 color per slice
    - Hover tooltip showing "label: value (percentage%)" on each arc path
    - Title if config.title

    **LineChart.ts** — `renderLineChart(container: HTMLDivElement, data: { label: string; value: number }[], config: ChartConfig): void`
    - Aspect ratio: 5:3 (height = width * 0.6)
    - Margins: top 20, right 20, bottom 40, left 50
    - d3.scalePoint for x-axis (categorical labels), d3.scaleLinear for y-axis
    - d3.line() path with `.curve(d3.curveMonotoneX)` for smooth curves
    - Data circles at each point (r=3), line path, both with 300ms transitions
    - Fill: config.color or `d3.schemeTableau10[0]` for line stroke and circle fill
    - Hover tooltip on circles showing "label: value"
    - X-axis: `.ticks(5)` for label density, y-axis: `d3.format('~s')`
    - Title if config.title

    **ScatterChart.ts** — `renderScatterChart(container: HTMLDivElement, data: { x: number; y: number }[], config: ChartConfig): void`
    - Aspect ratio: 1:1 (square)
    - Margins: top 20, right 20, bottom 40, left 50
    - d3.scaleLinear for both axes
    - Circles (r=4) positioned at (x, y) with 300ms transitions
    - D3 data join with `.data(data, d => `${d.x},${d.y}`)` key function
    - Fill: config.color or `d3.schemeTableau10[0]`, opacity 0.7
    - Hover tooltip showing "x: xValue, y: yValue"
    - Axis labels from config.x and config.y field names
    - Title if config.title
