```swift
enum Tier: String, Comparable, CaseIterable {
    case free, pro, workbench

    static func < (lhs: Tier, rhs: Tier) -> Bool {
        let order: [Tier] = [.free, .pro, .workbench]
        return order.firstIndex(of: lhs)! < order.firstIndex(of: rhs)!
    }
}
```

**SubscriptionManager class:**

- `@Published private(set) var currentTier: Tier = .free`
- `@Published private(set) var products: [Product] = []` -- sorted by price ascending after loading
- `@Published private(set) var purchaseInProgress: Bool = false` -- UI loading state

**Product IDs** (must match .storekit config exactly):

static let productIDs: Set<String> = [

**init():**
1. Start `Transaction.updates` listener FIRST (per Pitfall 6 from RESEARCH.md -- don't miss pending transactions)
2. Then `Task { await loadProducts() }`
3. Then `Task { await refreshEntitlements() }`

**deinit:** Cancel the updates task

**loadProducts():**
- `products = try await Product.products(for: Self.productIDs).sorted { $0.price < $1.price }`
- Log error but don't throw -- app functions without products loaded

**purchase(_ product: Product) async throws -> Bool:**
- Set `purchaseInProgress = true`, defer reset to false
- Call `product.purchase()`
- On `.success(let verification)`: call `checkVerified(verification)`, then `refreshEntitlements()`, then `transaction.finish()` -- CRITICAL: finish() ONLY after updating entitlements (Pitfall from RESEARCH.md)
- On `.userCancelled`: return false
- On `.pending`: return false (Ask to Buy)
- On `@unknown default`: return false

**refreshEntitlements():**
- Iterate `Transaction.currentEntitlements`
- For each `.verified(let transaction)`: map `tierForProductID(transaction.productID)` and track highest
- For `.unverified`: skip (do NOT grant entitlement -- security)
- Set `currentTier = highestTier`

**restorePurchases():**
- Call `try await AppStore.sync()` (forces re-verification of all transactions)
- Then call `refreshEntitlements()`
- Per CONTEXT.md: Restore Purchases button needed for App Store review compliance

**listenForTransactionUpdates() -> Task<Void, Never>:**
- `Task(priority: .background)` iterating `Transaction.updates`
- For verified: `refreshEntitlements()` then `transaction.finish()`
- For unverified: skip

**checkVerified<T>(_ result: VerificationResult<T>) throws -> T:**
- `.verified(let value)`: return value
- `.unverified`: throw StoreError.failedVerification

**tierForProductID(_ productID: String) -> Tier:**
- If contains "workbench": return .workbench
- If contains "pro": return .pro
- Default: return .free

Add `import StoreKit` and `import os` at top. Use `Logger(subsystem: "works.isometry.app", category: "Subscriptions")` for logging.

**1. Create FeatureGate.swift:**

Create `native/Isometry/Isometry/FeatureGate.swift` with:

```swift
// FeatureGate.swift -- TIER-04
// Enforces tier restrictions before allowing native actions.

enum NativeFeature: String, CaseIterable {
    case fileImport     // Pro tier (per CONTEXT.md: "Pro gates ETL/file import")
    case cloudSave      // Pro tier (future -- extensible)
    case exportData     // Pro tier (future -- extensible)
}

struct FeatureGate {
    static func requiredTier(for feature: NativeFeature) -> Tier {
        switch feature {
        case .fileImport:  return .pro
        case .cloudSave:   return .pro
        case .exportData:  return .pro
        }
    }

    static func isAllowed(_ feature: NativeFeature, for currentTier: Tier) -> Bool {
        currentTier >= requiredTier(for: feature)
    }
}
```

Per CONTEXT.md decisions: "Pro tier gates ETL/file import features and future export". File import is Pro-gated. All 9 views are Free. iCloud sync is Free.
