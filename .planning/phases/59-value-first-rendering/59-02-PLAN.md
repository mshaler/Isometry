   - Triggered by hover (mouseenter), not click
   - Shows card NAMES (from `_cardNameCache` or `d.cardNames` as fallback), not raw IDs
   - Header: `"${d.count} cards"` (always plural since badge only appears for 2+ cards)

   - Tooltip positioning: anchored below the badge element (same anchorRect pattern as existing tooltip)
   - Tooltip class: `sg-overflow-tooltip` (distinct from `sg-supercard-tooltip`)
   - Tooltip mouseenter cancels the dismiss timer; tooltip mouseleave starts it

3. Create `_closeOverflowTooltip(): void` — removes tooltip element, clears timer.

4. In the `_renderCells` spreadsheet branch (inside the `if (d.count > 1)` block from Plan 01), attach event listeners to the badge element:
   ```typescript
   badge.addEventListener('mouseenter', () => {
     if (this._overflowTooltipTimer) {
       clearTimeout(this._overflowTooltipTimer);
       this._overflowTooltipTimer = null;
     }
     this._openOverflowTooltip(badge, d);
   });
   badge.addEventListener('mouseleave', () => {
     this._overflowTooltipTimer = setTimeout(() => {
       this._closeOverflowTooltip();
     }, 150);
   });
   ```

5. In `_openOverflowTooltip`, after appending the tooltip to `_rootEl`, add mouseenter/mouseleave on the tooltip itself:

   ```typescript
     this._overflowTooltipTimer = setTimeout(() => {
       this._closeOverflowTooltip();
     }, 150);
   });
   ```
