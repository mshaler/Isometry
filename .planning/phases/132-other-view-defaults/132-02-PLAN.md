3. Add a public `updateRecommendations(sourceType: string | null)` method to the `SidebarNav` class. This method:

a. Iterates over all items in the `visualization` section (keys: list, gallery, kanban, grid, supergrid, calendar, timeline, network, tree).

b. Calls `resolveRecommendation(sourceType)` once to get the recommendation (or null).

c. For each visualization item element in `this._itemEls`:
   - If `sourceType` is null or resolveRecommendation returns null: remove any existing `.sidebar-item__badge` span from the button, clear the `title` attribute on the button.
   - If recommendation exists AND the item's `data-item-key` matches `recommendation.recommendedView`: append a `<span class="sidebar-item__badge" aria-hidden="true" data-testid="sidebar-badge-recommended">✦</span>` after the label span, set `title` attribute on the button to `recommendation.tooltipText`.
   - If recommendation exists but item key does NOT match recommendedView: remove any existing badge, clear title.

d. Implementation detail: To avoid duplicating badge spans on repeated calls, always remove existing `.sidebar-item__badge` from the button before conditionally adding a new one.

```typescript
updateRecommendations(sourceType: string | null): void {
  const recommendation = sourceType ? resolveRecommendation(sourceType) : null;

  // Iterate all visualization section items
  for (const [compositeKey, btn] of this._itemEls) {
    if (!compositeKey.startsWith('visualization:')) continue;
    const itemKey = compositeKey.split(':')[1]!;

    // Remove existing badge (idempotent)
    const existingBadge = btn.querySelector('.sidebar-item__badge');
    if (existingBadge) existingBadge.remove();
    btn.removeAttribute('title');

    // Add badge if this item is the recommended view
    if (recommendation && itemKey === recommendation.recommendedView) {
      const badge = document.createElement('span');
      badge.className = 'sidebar-item__badge';
      badge.setAttribute('aria-hidden', 'true');
      badge.setAttribute('data-testid', 'sidebar-badge-recommended');
      badge.textContent = '\u2736'; // ✦ U+2736 SIX POINTED BLACK STAR
      btn.appendChild(badge);
      btn.setAttribute('title', recommendation.tooltipText);
    }
  }
}
```

4. In the `destroy()` method, no additional cleanup is needed — badge spans are child nodes of button elements that are already cleaned up when `_itemEls` is cleared and `_navEl` is removed.
