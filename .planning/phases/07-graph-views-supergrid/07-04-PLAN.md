  <behavior>
    - buildHeaderCells(axisValues, collapsedSet) returns header cells organized by level
    - Each header cell has value, level, colStart (1-based), colSpan, isCollapsed
    - Single-level: each unique value gets colSpan=1
    - Two-level: parent header spans the count of its distinct children
    - Three-level: grandparent spans total leaf count of all children and grandchildren
    - Collapsed headers: children are hidden, parent colSpan is reduced to 1
    - Empty axis values (no data at that intersection) still produce header cells
    - buildGridTemplateColumns(leafCount) returns CSS grid-template-columns string
    - Cardinality guard: if leaf count > MAX_LEAF_COLUMNS (50), excess values collapsed to "Other"

2. Create `src/views/supergrid/SuperStackHeader.ts`:

**Core types:**
```typescript
export interface HeaderCell {
  /** Axis value label displayed in the header */
  value: string;
  /** Nesting level: 0 = outermost (primary), 1 = secondary, 2 = tertiary */
  level: number;
  /** 1-based CSS grid-column-start position */
  colStart: number;
  /** CSS grid-column: span N — number of leaf columns this header covers */
  colSpan: number;
  /** Whether the user has collapsed this header group */
  isCollapsed: boolean;
}
```

**MAX_LEAF_COLUMNS = 50** — cardinality explosion guard.

**collapseToOther(values: string[], max: number): string[]:**
- If values.length <= max, return as-is
- Otherwise, return first (max - 1) values + 'Other'

**buildHeaderCells(axisValues: string[][], collapsedSet: Set<string>):**

Algorithm from RESEARCH.md with refinements:
1. `axisValues` is an array of tuples, each representing one leaf column: `[primaryVal, secondaryVal?, tertiaryVal?]`
2. The number of levels = tuple length (1, 2, or 3)
3. Walk through leaves left-to-right
4. For each level, detect run-length of consecutive identical values
5. When a value changes, emit a HeaderCell with colSpan = run length
6. For collapsed parents (checked via `collapsedSet.has('level:value')`): the parent cell gets colSpan=1 and all child cells under it are omitted (they don't appear in the output)
7. Return `{ headers: HeaderCell[][], leafCount: number }`

**buildGridTemplateColumns(leafCount: number, rowHeaderWidth = 160):**
```typescript
return `${rowHeaderWidth}px repeat(${leafCount}, minmax(60px, 1fr))`;
```

**applyCardinalityGuard(axisValues: string[][]): string[][]:**
- Check distinct values at each level
- If total leaf count > MAX_LEAF_COLUMNS, collapse the innermost level's least frequent values into "Other"
- Return modified axisValues

2. Create `src/views/SuperGrid.ts`:

**Constructor:**
- Accepts ViewConfig + optional PAFVProvider reference (for reading stacked axis config)
- Internal state:
  - `collapsedSet: Set<string>` — collapsed headers, keyed by 'level:value'
  - `gridContainer: HTMLDivElement | null` — the CSS Grid container
  - `colAxes: AxisMapping[]`, `rowAxes: AxisMapping[]` — axis configuration

**mount(container):**
1. Create root `div.supergrid-view.view-root`
2. Create inner `div.supergrid-container` with `display: grid`
3. The grid template will be set dynamically in render()

**render(cards):**
1. Determine axis configuration:
   - Read from PAFVProvider state if available
   - Default: column axis = `card_type`, row axis = `folder` (reasonable defaults)
   - SuperGrid supports up to 3 stacked axes per dimension
2. Extract distinct values per axis from the cards data:
   - For each column axis field, collect unique values (sorted)
   - Apply cardinality guard (max 50 leaf columns)
3. Build axis value tuples (each leaf column = one tuple)
4. Call `buildHeaderCells(axisValues, this.collapsedSet)` to compute headers
5. Set `gridTemplateColumns` via `buildGridTemplateColumns(leafCount)`
6. Set `gridTemplateRows` based on header levels + data row count
7. Render column headers: for each level, create div elements with `grid-column: start / span N`
8. Render row headers similarly for row dimension
9. Render data cells at each (row, col) intersection:
   - Find cards matching this intersection (filter cards where all axis values match)
   - Display count badge (e.g., "3") — Claude's discretion: using count badges for <16ms performance
   - Empty cells get class 'empty-cell' with subtle border
   - D3 data join with key function: `d => d.rowKey + ':' + d.colKey`

**Header collapse:**
- Click handler on each header div
- On click: toggle 'level:value' in collapsedSet
- Re-call render() to rebuild grid with updated collapse state
- The rebuild is efficient because buildHeaderCells handles the collapse math

**CSS Grid styling (inline or classes):**
- Headers: bold text, centered, visible bottom border for each level
- Data cells: centered count badge, subtle border
- Empty cells: same border, lighter background (e.g., rgba(255,255,255,0.02))
- Grid gap: 1px (visible grid lines)

**destroy():**
- Remove grid container from parent
- Clear collapsedSet
- Null out references
