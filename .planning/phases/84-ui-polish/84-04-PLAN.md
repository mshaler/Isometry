# Plan 84-04: Keyboard navigation for CommandBar menu and ViewTabBar

## Objective

CommandBar declares `role="menu"` but only handles Escape. ViewTabBar declares `role="tablist"` but only wires click handlers. Both violate ARIA keyboard contracts. Implement roving tabindex navigation patterns for both components.

<tasks>

<task id="1">
**CommandBar: Add roving tabindex helpers**

Add private methods to CommandBar.ts:

```typescript
private _getMenuItems(): HTMLElement[] {
  return Array.from(
    this._dropdownEl?.querySelectorAll<HTMLElement>('[role="menuitem"]') ?? []
  );
}

private _focusItem(index: number): void {
  const items = this._getMenuItems();
  items.forEach((el, i) =>
    el.setAttribute('tabindex', i === index ? '0' : '-1')
  );
  items[index]?.focus();
}

private _moveFocus(delta: 1 | -1): void {
  const items = this._getMenuItems();
  const current = items.indexOf(document.activeElement as HTMLElement);
  const next = (current + delta + items.length) % items.length;
  this._focusItem(next);
}
```

Ensure _createMenuItem() sets `role="menuitem"` and `tabindex="-1"` on each item.
</task>

<task id="2">
**CommandBar: Focus first item on open, extend keydown handler**

In `_openDropdown()`, after showing menu:
```typescript
const items = this._getMenuItems();
items.forEach((el, i) => el.setAttribute('tabindex', i === 0 ? '0' : '-1'));
items[0]?.focus();
```

Extend `_onDocumentKeydown` dropdown-open block:
```typescript
case 'ArrowDown': e.preventDefault(); this._moveFocus(1); break;
case 'ArrowUp':   e.preventDefault(); this._moveFocus(-1); break;
case 'Home':      e.preventDefault(); this._focusItem(0); break;
case 'End':       e.preventDefault(); this._focusItem(this._getMenuItems().length - 1); break;
case 'Escape':    this._closeDropdown(); this._triggerEl?.focus(); break;
```
</task>

<task id="4">
**ViewTabBar: Arrow key tab switching**

Add keydown listener on the nav element:
```typescript
this._el.addEventListener('keydown', (e: KeyboardEvent) => {
  const types = VIEW_LABELS.map((v) => v.type);
  const currentIndex = types.indexOf(this._activeType);
  let nextIndex: number | null = null;
  if (e.key === 'ArrowRight') nextIndex = (currentIndex + 1) % types.length;
  else if (e.key === 'ArrowLeft') nextIndex = (currentIndex - 1 + types.length) % types.length;
  else if (e.key === 'Home') nextIndex = 0;
  else if (e.key === 'End') nextIndex = types.length - 1;

  if (nextIndex !== null) {
    e.preventDefault();
    config.onSwitch(types[nextIndex]!);
    this._buttons.get(types[nextIndex]!)?.focus();
  }
});
```
</task>
