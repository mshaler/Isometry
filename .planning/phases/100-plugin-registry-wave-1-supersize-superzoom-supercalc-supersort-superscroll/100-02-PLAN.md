    - Export constants: `SCROLL_BUFFER = 2`, `VIRTUALIZATION_THRESHOLD = 100`
    - Export pure function:
      ```typescript
      export function getVisibleRange(
        scrollTop: number, rowHeight: number, containerHeight: number, totalRows: number
      ): { startRow: number; endRow: number }
      ```
      Port logic from `SuperGridVirtualizer.getVisibleRange()`:
      - `firstVisible = Math.floor(scrollTop / rowHeight)`
      - `lastVisible = Math.ceil((scrollTop + containerHeight) / rowHeight)`
      - `startRow = Math.max(0, firstVisible - SCROLL_BUFFER)`
      - `endRow = Math.min(totalRows, lastVisible + SCROLL_BUFFER)`
      - Return `{ startRow, endRow }`

    - Internal state: `_lastRange: { startRow: number, endRow: number } | null = null`

    Factory `createSuperScrollVirtualPlugin()`:
    - `transformData(cells, ctx)`:
      - Count unique rowIdx values to get totalRows
      - If totalRows <= VIRTUALIZATION_THRESHOLD, return cells unchanged (no windowing)
      - Get scrollTop from ctx.scrollTop, rowHeight from `layout.cellHeight` (cast ctx to RenderContextWithLayout)
      - Compute containerHeight: read from `ctx.rootEl.closest('.pv-scroll-container')?.clientHeight ?? 600`
      - Compute visible range via getVisibleRange()
      - If range equals `_lastRange`, return cells unchanged (avoid re-sort)
      - Filter cells: `cells.filter(c => c.rowIdx >= range.startRow && c.rowIdx < range.endRow)`
      - Store `_lastRange = range`
      - Return filtered cells

    - `afterRender(root)`:
      - Create or update two sentinel spacer divs:
        - `<div class="pv-scroll-sentinel-top">` with `height: ${_lastRange?.startRow * rowHeight}px`
        - `<div class="pv-scroll-sentinel-bottom">` with `height: ${(totalRows - _lastRange?.endRow) * rowHeight}px`
      - Insert sentinels at top/bottom of the scroll container's tbody (or equivalent data area)

    - `onScroll(scrollLeft, scrollTop, ctx)`:
      - Recompute visible range
      - If range changed from `_lastRange`, call a stored re-render callback (or set a flag for next transformData)

    - `destroy()`: Remove sentinel elements, reset state
