    **SuperSizeColResize.ts** (~120 LOC):
    - Export constants: `MIN_COL_WIDTH = 48`, `AUTO_FIT_PADDING = 24`, `AUTO_FIT_MAX = 400`
    - Internal state: `_colWidths: Map<number, number>`, `_dragging: { colIdx: number, startX: number, startWidth: number, pointerId: number } | null`
    - `onPointerEvent('pointerdown', e)`: Check if target has `.pv-resize-handle--width` class. If so, record drag start state with `e.pointerId`, set `document.body.style.cursor = 'col-resize'`, call `(e.target as HTMLElement).setPointerCapture(e.pointerId)`. Return true to consume.
    - `onPointerEvent('pointermove', e)`: If `_dragging`, compute `newWidth = Math.max(MIN_COL_WIDTH, _dragging.startWidth + (e.clientX - _dragging.startX))`. Store in `_colWidths.set(_dragging.colIdx, newWidth)`. Return true.
    - `onPointerEvent('pointerup', e)`: If `_dragging`, clear drag state, reset body cursor. Return true.
    - `onPointerEvent('dblclick', e)`: If target has `.pv-resize-handle--width`, measure max content width of column cells via `querySelectorAll('[data-col="${colIdx}"]')`, set width to `Math.min(measured + AUTO_FIT_PADDING, AUTO_FIT_MAX)`. Return true.
    - `transformLayout(layout)`: For each entry in `_colWidths`, set `layout.colWidths.set(colIdx, width)`. Return layout.

    - `destroy()`: Clear `_colWidths`, null out `_dragging`.
    - Shift+drag: If `e.shiftKey` during pointermove, set ALL colWidths to the same value as dragged column.

    - `transformLayout(layout)`: Apply scale: `layout.cellWidth *= _scale`, `layout.cellHeight *= _scale`. Return layout.

    **SuperZoomWheel.ts** (~100 LOC):
    - Export constants: `ZOOM_MIN = 0.5`, `ZOOM_MAX = 3.0`, `ZOOM_DEFAULT = 1.0`
    - Export pure functions ported verbatim from `src/views/supergrid/SuperZoom.ts`:
      - `normalizeWheelDelta(e: WheelEvent): number` — deltaMode normalization, cap +-24
      - `wheelDeltaToScaleFactor(dy: number): number` — asymmetric formula with WHEEL_SCALE_SPEEDUP = 2.0
    - Internal shared state object: `{ zoom: number, listeners: Set<(zoom: number) => void> }`
    - Factory `createSuperZoomWheelPlugin(zoomState: { zoom: number, listeners: Set<(z: number) => void> })`:
      - Store zoom in shared `zoomState` object so slider can sync
      - `onPointerEvent`: NOT used for wheel — wheel events go through a separate listener
      - Instead, use `afterRender` to attach a non-passive wheel listener on `root.closest('.pv-scroll-container') || root.parentElement` with `{ passive: false }`
      - Wheel handler: `if (!e.ctrlKey) return;` → `e.preventDefault()` → compute new zoom via `zoomState.zoom * wheelDeltaToScaleFactor(normalizeWheelDelta(e))` → clamp to [ZOOM_MIN, ZOOM_MAX] → store in `zoomState.zoom` → notify listeners
      - `transformLayout(layout)`: Set `layout.zoom = zoomState.zoom`. Apply zoom to `layout.cellWidth *= zoomState.zoom` and `layout.cellHeight *= zoomState.zoom`. Return layout.
      - Cmd+0 reset: attach keydown listener on document for `e.metaKey && e.key === '0'` → set zoom to ZOOM_DEFAULT → notify
      - `destroy()`: Remove wheel + keydown listeners

        - Set `accent-color: var(--pv-accent)` on slider for themed thumb
      - Register as zoom state listener to update slider position when wheel zoom changes
      - `destroy()`: Remove zoom control container, unregister listener
