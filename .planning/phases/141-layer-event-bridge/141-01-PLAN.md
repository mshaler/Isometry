<objective>
Wire Layer 1 data cells to the plugin pipeline: add data-key/data-row/data-col attributes during render, bridge pointer events from the scroll container to PluginRegistry.runOnPointerEvent with a real RenderContext, change afterRender rootEl from the overlay to the scroll container so SuperSelect/SuperAudit can find data cells, and apply user-select:none to prevent text selection during drag.

Purpose: SuperSelect and SuperAudit plugins already read data-key/data-row/data-col and query .pv-data-cell within the rootEl they receive, but currently (a) those attributes are never set on cells, (b) pointer events only fire from the overlay (Layer 2), and (c) afterRender receives the overlay as rootEl where no data cells exist. This plan closes all three gaps.

Output: Modified PivotGrid.ts, pivot.css, and a test file verifying the wiring.
</objective>
