## Spacing Scale

All values from `--space-*` tokens in `design-tokens.css`. No tailwind; no 8-point scale — project uses a 4/8/12/16/24 compact scale tuned for dense data UIs.

| Token | Value | Usage |
|-------|-------|-------|
| --space-xs | 4px | Icon gaps, badge padding, checkbox gap |
| --space-sm | 8px | Button padding (vertical), row gap, section gap |
| --space-md | 12px | Default element spacing, dialog flex gap |
| --space-lg | 16px | Button padding (horizontal), toast padding |
| --space-xl | 24px | Dialog padding, section breaks |

Exceptions:
- Touch targets (checkboxes): min 20×20px — already established in `.disc-row input[type="checkbox"]`
- Progress bar height: 3px — intentional below-scale decorative element (from `.import-toast-progress`)
- Dialog max-height list area: 320px — scroll container cap (from `.disc-sheet__list`)

Source: `src/styles/design-tokens.css`, `src/styles/directory-discovery.css`

## Interaction States

### DirectoryDiscoverySheet (extends Phase 123 component)

The existing `.disc-sheet` modal handles checkbox selection (IMPT-01). Phase 124 adds the import execution path. No new dialog is needed — the "Import Selected" CTA in the existing sheet triggers import.

**State machine for DirectoryDiscoverySheet:**

| State | Visual Indicator | Trigger |
|-------|-----------------|---------|
| Idle — 0 checked | "Import Selected" button disabled, opacity 0.5, cursor not-allowed | All checkboxes unchecked |
| Idle — 1+ checked | "Import Selected" button enabled, opacity 1.0, cursor pointer | Any checkbox checked |
| Importing | Button replaced with spinner + "Importing…" label, checkboxes disabled, cancel button hidden | User clicks "Import Selected" |
| Per-directory progress | Directory name + card count in status line, progress bar fills left-to-right | Each directory completes |
| All complete | "Done — N cards imported" heading, "Close" button replaces actions row | All directories finish |
| Error | Error row inlined below failed directory, other directories continue, "Close" button available | Import error for a directory |

### Import Progress Display (IMPT-04)

Progress is shown inline within the `.disc-sheet` modal — NOT a separate toast. The existing `.import-toast` pattern is for single-file imports; multi-directory progress needs richer per-row status. Layout:

```
┌─ .disc-sheet ──────────────────────────────┐
│  Importing Alto-Index Directory            │  ← .disc-sheet__title (text updates)
│  3 of 5 directories complete               │  ← .disc-sheet__subtitle (live count)
│  ─────────────────────────────────────    │  ← .disc-import-progress-bar (accent fill)
│  ✓  notes          823 cards    [done]     │  ← .disc-import-row (state variants)
│  ✓  contacts        41 cards    [done]     │
│  ◌  calendar         — cards    [active]   │  ← spinner on active row
│  ·  messages                   [pending]  │
│  ·  books                      [pending]  │
│  ─────────────────────────────────────    │
│                            [Close]         │  ← only button while importing; enabled when done
└────────────────────────────────────────────┘
```

**Import row states (`.disc-import-row`):**

| State | Indicator | CSS |
|-------|-----------|-----|
| pending | muted name, no count | color: --text-muted |
| active | name at full opacity, spinning indicator, "processing…" | color: --text-primary + CSS spinner |
| done | checkmark prefix ("✓"), card count visible | color: --text-primary |
| error | "✗" prefix, error message below in --danger color | color: --danger |

**Progress bar (`div.disc-import-progress-bar`):**
- Height: 4px
- Background track: --border-subtle
- Fill: --accent
- Width animates via inline `style="width: N%"` updated by JS
- `transition: width var(--transition-fast)` (150ms)

### Checkbox Interaction (IMPT-01)

Inherits full behavior from Phase 123 `.disc-row`. All directories pre-checked on open. Clicking anywhere on the `.disc-row` (except directly on checkbox) toggles the checkbox. "Import Selected" button disabled when zero directories checked.

**Select All / Deselect All control:**

A single `.disc-sheet__toggle-all` link-style button above the list.
- Label: "Deselect All" when all are checked (default state)
- Label: "Select All" when any are unchecked
- Styled as `font-size: var(--text-sm); color: var(--accent); background: none; border: none; cursor: pointer`
- Updates instantly on each checkbox change

## Accessibility Contract

All items from the existing WCAG 2.1 AA commitment apply. Phase 124 additions:

- `aria-live="polite"` region on `.disc-sheet__subtitle` — updates as each directory completes
- Spinner element has `aria-label="processing"` and `role="status"`
- Each `.disc-import-row` has `aria-label="{dirName}: {status}"` updated as state changes
- Progress bar: `role="progressbar" aria-valuenow="{percent}" aria-valuemin="0" aria-valuemax="100" aria-label="Import progress"`
- While importing, "Close" button is enabled but labeled "Close (import in progress)" until all directories complete, then label simplifies to "Close"
- Escape key during import does NOT cancel the in-progress import — it is ignored while `state === 'importing'`
