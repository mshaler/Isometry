**Important SVG note:** D3 `.attr('fill', ...)` and `.attr('stroke', ...)` on SVG elements accept CSS `var()` references when set via the `style` attribute. However, when set via SVG presentation attributes (`.attr('fill', ...)`), `var()` works in modern browsers (Chrome 85+, Safari 15+, Firefox 92+). For this project's target (Safari 17+ / macOS 14+), `var()` in SVG attributes is fully supported. Use `.attr('fill', 'var(--token)')` directly.
