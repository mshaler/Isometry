# Plan 114-02: Preview Tab Integration

## Objective

<objective>
Complete Preview tab integration by adding tab state persistence across sessions and verifying view-specific PAFV controls per tab. Ensure seamless user experience when switching between SuperGrid, Network, and Timeline tabs.
</objective>

## Context

The Preview pane already has working tab bar with SuperGrid, Network, Timeline, Data Inspector, Web Preview, and D3 Viz tabs. However:

1. **Tab state not persisted:** Refreshing or returning to the app resets to default tab
2. **PAFV controls per tab:** Need to verify each view has appropriate axis controls

## Tasks

### Task 1: Add Tab State Persistence

Update `src/components/notebook/PreviewComponent.tsx` to persist active tab:

**Implementation:**
```typescript
// Initialize from sessionStorage
const [activeTab, setActiveTab] = useState<PreviewTab>(() => {
  const stored = sessionStorage.getItem('preview-active-tab');
  return (stored as PreviewTab) || 'supergrid';
});

// Persist on change
useEffect(() => {
  sessionStorage.setItem('preview-active-tab', activeTab);
}, [activeTab]);
```
