```css
/* ---------------------------------------------------------------------------
 * SVG text reset — prevents HTML container styles from leaking into SVG.
 * SVG <text> inherits CSS letter-spacing, text-transform, and word-spacing
 * from parent HTML elements, causing garbled axis labels in D3 charts and
 * histogram scrubbers (especially in Safari/WebKit).
 * This reset is global to protect all current and future SVG text.
 * Requirement: BUGF-01
 * -------------------------------------------------------------------------- */
svg text {
  letter-spacing: normal;
  text-transform: none;
  word-spacing: normal;
}
```

Do NOT modify any of the 6 existing letter-spacing declarations in other CSS files — they are intentional styling on HTML elements (section headers, badges, labels). The SVG reset prevents inheritance without removing HTML styling.

Do NOT consolidate existing letter-spacing values into design tokens — the SVG fix does not depend on it.
