## Goal

Add schema-aware field migration to `StateManager.restore()` so that persisted provider state referencing columns that no longer exist in the database schema is silently pruned before reaching provider `setState()`. FilterProvider and PAFVProvider both receive clean state with only valid field references.

## must_haves

1. StateManager has a `setSchemaProvider(sp)` setter and a private `_migrateState()` method called inside `restore()` between `JSON.parse` and `provider.setState()`
2. Filter state migration prunes: filters[] entries with unknown field, axisFilters{} entries with unknown keys, rangeFilters{} entries with unknown keys
3. PAFV state migration: nulls xAxis/yAxis/groupBy with unknown field, filters colAxes[]/rowAxes[] entries with unknown field, passes colWidths/sortOverrides/collapseState unchanged
4. When SchemaProvider is not wired or not initialized, `_migrateState()` is a pass-through (no pruning)
5. `main.ts` calls `sm.setSchemaProvider(schemaProvider)` before `sm.restore()` -- wait, StateManager is not currently wired in main.ts. We need to wire it.
6. Integration tests cover: filter pruning (valid survives, invalid dropped), axisFilters/rangeFilters per-key pruning, PAFV axis nulling, PAFV array filtering, colWidths/sortOverrides/collapseState survival, no-schema pass-through
7. All existing StateManager tests continue to pass

<title>Add setSchemaProvider() and _migrateState() to StateManager</title>
<type>code</type>
<files>src/providers/StateManager.ts</files>
<details>
Add the following to `StateManager`:

1. Import `SchemaProvider` type from `./SchemaProvider`.
2. Add private field `_schema: SchemaProvider | null = null`.
3. Add public `setSchemaProvider(sp: SchemaProvider | null): void` setter.
4. Add private `_migrateState(key: string, state: unknown): unknown` method:
   - Guard: `if (!this._schema?.initialized) return state;`
   - Guard: `if (typeof state !== 'object' || state === null) return state;`
   - Route by key: `'filter'` -> `_migrateFilterState()`, `'pafv'` -> `_migratePAFVState()`
   - Default: return state unchanged (alias and all other keys pass through)
5. Add private `_migrateFilterState(state: unknown): unknown`:
   - Cast to `Record<string, unknown>`
   - Create helper `isValid = (f: string) => this._schema!.isValidColumn(f, 'cards')`
   - Prune `filters[]`: filter array entries where `typeof entry.field === 'string' && isValid(entry.field)`
   - Prune `axisFilters{}`: Object.fromEntries filtering unknown keys
   - Prune `rangeFilters{}`: Object.fromEntries filtering unknown keys
   - Return spread `{ ...s, filters, axisFilters, rangeFilters }`
6. Add private `_migratePAFVState(state: unknown): unknown`:
   - Cast to `Record<string, unknown>`
   - Create helper `isValid = (f: string) => this._schema!.isValidColumn(f, 'cards')`
   - `nullIfInvalid(axis)`: if axis is null/undefined return as-is; if axis.field is unknown return null; else return axis
   - `filterAxes(axes)`: if array, filter entries where axis.field is valid; else return as-is
   - Return spread with: xAxis/yAxis/groupBy -> nullIfInvalid, colAxes/rowAxes -> filterAxes, all other fields (colWidths, sortOverrides, collapseState, viewType, aggregation) pass through unchanged
7. Modify `restore()`: between `JSON.parse(row.value)` and `provider.setState(parsed)`, insert `const migrated = this._migrateState(row.key, parsed);` and pass `migrated` to `setState`.
</details>
