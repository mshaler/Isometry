**Fix 1: Footer label format.**

In `SuperCalcFooter.ts` `afterRender`, the footer currently shows a glyph (`GLYPHS[colCfg.fn]`) + raw number. The UI-SPEC requires label format: `"{FUNCTION} of {field}"` e.g. "SUM of weight" — but only in the label column (first column), not in every aggregate cell.

The current implementation puts one `.pv-calc-cell` per visible column. There is no "label column" — each cell shows glyph + value. To match UI-SPEC:

Add a label cell at the start of the footer (before the per-column cells):

```typescript
// In afterRender, after creating/clearing footer and before the column loop:
// Determine the aggregate label text
const firstColCfg = getColConfig(config, 0);
const labelCell = document.createElement('div');
labelCell.className = 'pv-calc-label';
labelCell.style.cssText = 'color:var(--text-muted);font-size:11px;font-family:var(--font-mono);white-space:nowrap;';

// Get the column dimension name for the label
const colDimNames = ctx.colDimensions.map(d => d.name).join(', ');
if (firstColCfg.fn !== 'NONE') {
  labelCell.textContent = `${firstColCfg.fn} of ${colDimNames}`;
}
footer.appendChild(labelCell);
```

Actually, re-reading the UI-SPEC more carefully: the label format is "per column" — `"{FUNCTION} of {field}"`. Each column may have a different aggregate function (per CalcConfig). The label goes in the glyph span area, replacing the glyph-only approach.

**Revised approach:** Keep the existing per-column cell structure. Update each cell to show the label format when the cell has a value:

In the cell rendering loop (around line 266-270 of SuperCalcFooter.ts), replace the glyph + raw number with the label format:

```typescript
} else if (result.value !== null) {
  // Format: "{FUNCTION} of {field}: {value}"
  // But since cells are narrow, use tooltip for full label
  // Cell shows: glyph + formatted value (existing), add title attribute for full label
  const glyph = document.createElement('span');
  glyph.className = 'pv-calc-glyph';
  glyph.textContent = GLYPHS[colCfg.fn];
  cell.appendChild(glyph);

  const valSpan = document.createElement('span');
  valSpan.textContent = ` ${typeof result.value === 'number' ? result.value.toLocaleString() : result.value}`;
  cell.appendChild(valSpan);

  // Full label as title tooltip per UI-SPEC
  const colDimName = ctx.colDimensions.length > 0 ? ctx.colDimensions[ctx.colDimensions.length - 1]?.name ?? '' : '';
  cell.title = `${colCfg.fn} of ${colDimName}`;
}
```
