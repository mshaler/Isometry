   - This array is the source of truth for arrow key navigation order (section headers excluded)

3. **Add private field for current focus index:**

4. **Add roving tabindex initialization (end of `mount()`):**
   - Set all dock item buttons to `tabindex="-1"`
   - Set first item to `tabindex="0"` (the initial roving focus target)
   ```typescript
   this._orderedItems.forEach((el, i) => el.setAttribute('tabindex', i === 0 ? '0' : '-1'));
   ```

5. **Add keydown handler on nav element (event delegation):**
   ```typescript
   this._keydownHandler = (e: KeyboardEvent) => {
     const target = e.target as HTMLElement;
     if (!target.classList.contains('dock-nav__item')) return;
     const items = this._orderedItems;
     const len = items.length;
     if (len === 0) return;

     let nextIndex = this._focusIndex;
     if (e.key === 'ArrowDown') {
       e.preventDefault();
       nextIndex = (this._focusIndex + 1) % len;
     } else if (e.key === 'ArrowUp') {
       e.preventDefault();
       nextIndex = (this._focusIndex - 1 + len) % len;
     } else if (e.key === 'Home') {
       e.preventDefault();
       nextIndex = 0;
     } else if (e.key === 'End') {
       e.preventDefault();
       nextIndex = len - 1;
     } else {
       return;
     }

     items[this._focusIndex]!.setAttribute('tabindex', '-1');
     items[nextIndex]!.setAttribute('tabindex', '0');
     items[nextIndex]!.focus();
     this._focusIndex = nextIndex;
   };
   nav.addEventListener('keydown', this._keydownHandler);
   ```
   Store `_keydownHandler` as a private field for cleanup in `destroy()`.

6. **Update `_setActive()` to manage `aria-selected`:**
   - When deactivating previous: `prevEl.setAttribute('aria-selected', 'false');`
   - When activating new: `itemEl.setAttribute('aria-selected', 'true');`
   (Keep existing `aria-current="page"` logic as-is per D-13.)

7. **Update `_applyCollapseState()` to manage tabindex per Hidden state (per D-09):**
   ```typescript
   if (state === 'hidden') {
     // Remove all dock items from tab order
     this._orderedItems.forEach(el => el.setAttribute('tabindex', '-1'));
   } else {
     // Restore roving tabindex — current focus index gets tabindex="0"
     this._orderedItems.forEach((el, i) => el.setAttribute('tabindex', i === this._focusIndex ? '0' : '-1'));
   }
   ```

8. **Update `destroy()` to clean up keydown handler:**
   ```typescript
   if (this._navEl && this._keydownHandler) {
     this._navEl.removeEventListener('keydown', this._keydownHandler);
   }
   this._keydownHandler = null;
   this._orderedItems = [];
   ```
