**Modify `src/ui/DockNav.ts`:**

1. Add import at top: `import { renderMinimap, clearMinimap } from './MinimapRenderer';`

2. Add new private fields to DockNav class:
   - `private _idleCallbackIds: number[] = [];` — tracks pending requestIdleCallback handles for cancellation
   - `private _thumbnailDataSource: (() => { cards: CardDatum[]; pafvAxes: { xAxis: AxisMapping | null; yAxis: AxisMapping | null; groupBy: AxisMapping | null; colAxes: AxisMapping[]; rowAxes: AxisMapping[] } }) | null = null;` — data callback injected from main.ts

3. Add public method `setThumbnailDataSource(fn: typeof this._thumbnailDataSource): void` that stores the callback. This is the seam for main.ts to provide current cards and PAFV state without DockNav subscribing to StateCoordinator directly (per STATE.md constraint: "MinimapRenderer must be lazy-on-hover only -- never subscribed to StateCoordinator").

4. Modify `_applyCollapseState(state: CollapseState)` — after the existing tabindex management block, add:
   ```typescript
   // Cancel any pending thumbnail renders
   for (const id of this._idleCallbackIds) cancelIdleCallback(id);
   this._idleCallbackIds = [];

   if (state === 'icon-thumbnail') {
     this._renderAllThumbnails();
   }
   ```

5. Add private method `_renderAllThumbnails(): void`:
   - If `_thumbnailDataSource` is null, return early
   - Get data: `const { cards, pafvAxes } = this._thumbnailDataSource();`
   - Collect all visualization item buttons: iterate `_itemEls` entries, filter where sectionKey === 'visualize' (split composite key on ':' and check first part)
   - Stagger rendering in batches of 3 via requestIdleCallback:
     ```typescript
     const vizItems = [...this._itemEls.entries()].filter(([key]) => key.startsWith('visualize:'));
     const BATCH_SIZE = 3;
     for (let i = 0; i < vizItems.length; i += BATCH_SIZE) {
       const batch = vizItems.slice(i, i + BATCH_SIZE);
       const id = requestIdleCallback(() => {
         for (const [compositeKey, btn] of batch) {
           const viewKey = compositeKey.split(':')[1]!;
           const thumbEl = btn.querySelector<HTMLDivElement>('.dock-nav__item-thumb');
           if (thumbEl) renderMinimap(thumbEl, viewKey, cards, pafvAxes);
         }
       });
       this._idleCallbackIds.push(id);
     }
     ```

6. In `destroy()` method, add cleanup: cancel all idle callbacks and clear the array, set `_thumbnailDataSource = null`.
