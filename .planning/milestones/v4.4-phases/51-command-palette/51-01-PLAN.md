  <action>
    **1. Create `src/palette/fuzzy.ts`:**

    Export a `fuzzyMatch(query: string, target: string): number | null` function.

    Scoring algorithm (from RESEARCH.md):
    - Empty query returns null
    - Case-insensitive comparison (lowercase both)
    - Substring match shortcut: if target includes query, score = 1000 - target.length + (starts-with bonus of 100)
    - Character-by-character fuzzy: iterate target chars, match against query chars in order. Score consecutive matches with increasing bonus (consecutive * 2). Word boundary bonus (+10) when match follows space/hyphen or is at position 0.
    - Return null if not all query chars were matched

    **2. Create `src/palette/CommandRegistry.ts`:**

    Export `PaletteCommand` interface:
    ```typescript
    export interface PaletteCommand {

      shortcut?: string;                       // e.g. 'Cmd+1' -- displayed as <kbd>
      icon?: string;                           // Category icon character

    ```

    Export `CommandRegistry` class:

    - `getById(id: string): PaletteCommand | undefined` -- lookup by ID
