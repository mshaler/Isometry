    - Context menu dismissed by click-outside (rAF-deferred) or Escape

    - data-axis-field attribute added to header elements in _renderCells() so context menu can identify which field was right-clicked
  </behavior>

    **Implementation:**

       - `_contextMenuEl: HTMLDivElement | null = null`
       - `_boundContextMenuOutsideClick: ((e: MouseEvent) => void) | null = null`

       - `_hiddenRows: Set<string> = new Set()`

       - In the header rendering loop (where `.col-header` / `.row-header` elements are created), add: `headerEl.dataset['axisField'] = levelAxisField`
       - This is needed so the context menu knows which field was right-clicked

    3. Register event delegation in `mount()` (ONE listener, not per-render — avoids Pitfall 2):
       ```
       this._boundContextMenuHandler = (e: MouseEvent) => {
         const header = (e.target as Element).closest<HTMLElement>('.col-header, .row-header');
         if (!header) return;
         e.preventDefault();
         e.stopPropagation();
         const axisField = header.dataset['axisField'] ?? '';
         if (!axisField) return;
         const dimension = header.classList.contains('col-header') ? 'col' : 'row';
         this._openContextMenu(e.clientX, e.clientY, axisField, dimension);
       };
       this._gridEl!.addEventListener('contextmenu', this._boundContextMenuHandler);
       ```

    4. Implement `_openContextMenu(clientX: number, clientY: number, field: string, dimension: 'col' | 'row')`:
       - Call `_closeContextMenu()` first
       - Compute position relative to `_rootEl` (same pattern as tooltip/filter dropdown):
         `const rootRect = _rootEl.getBoundingClientRect(); top = clientY - rootRect.top + _rootEl.scrollTop; left = clientX - rootRect.left + _rootEl.scrollLeft;`
       - Create menu div with class `sg-context-menu`, position absolute, z-index 30, background white, border, border-radius 6px, min-width 180px, shadow
       - Build context-aware items:

         e. **Separator**
         f. **Hide column** / **Show hidden (N)**: Check if field is in `_hiddenCols`/`_hiddenRows`. If hidden fields exist for this dimension, add "Show hidden (N)" item that clears the set and calls `_fetchAndRender()`. "Hide" item adds field to set and calls `_fetchAndRender()`.
       - Use createElement for all items (no innerHTML)
       - Each item: div with class `sg-context-menu-item`, hover background, padding
       - Append to `_rootEl`
       - rAF-deferred click-outside listener (same pattern as filter dropdown/tooltip)

    5. Implement `_closeContextMenu()`: remove from DOM, remove click-outside listener, null refs

    6. Wire hidden columns/rows into `_renderCells()`:
       - After computing cellPlacements (or in the header rendering), filter out any axes whose field is in `_hiddenCols`/`_hiddenRows`
       - This is Tier 3 ephemeral state — no persistence, resets on page reload

    8. Cleanup in `destroy()`: call `_closeContextMenu()`, remove contextmenu event listener from `_gridEl`

    - Do NOT call `_fetchAndRender()` from sort/filter menu items — route through provider setters which trigger coordinator subscription
