Add private fields to SuperGrid:
- `_filterDropdownEl: HTMLElement | null = null` — currently open dropdown
- `_boundFilterOutsideClick: ((e: MouseEvent) => void) | null = null` — stored for cleanup
- `_boundFilterEscapeHandler: ((e: KeyboardEvent) => void) | null = null` — stored for cleanup

Add private method `_createFilterIcon(axisField: string, dimension: 'col' | 'row'): HTMLSpanElement`:
- Creates `<span class="filter-icon">` with `data-filter-field` attribute
- Text content: `\u25BD` (hollow down triangle) when inactive, `\u25BC` (filled) when active
- Style: opacity 0/1 based on `this._filter.hasAxisFilter(axisField)`, accent color when active (`var(--sg-filter-active-color, #1a56f0)`), cursor pointer, margin-left 4px, font-size 10px, flex-shrink 0, user-select none, transition opacity 0.15s
- Click handler: `stopPropagation()`, `preventDefault()`, calls `this._openFilterDropdown(icon, axisField, dimension)`
- rAF-deferred parent hover-reveal: mouseenter → opacity 0.5 (if inactive), mouseleave → opacity 0 (if inactive). Same pattern as `_createSortIcon`.
- Filter icon placed AFTER sort icon in header flex container: `[grip][label][sort-icon][filter-icon]`

Add private method `_getAxisValues(axisField: string, dimension: 'col' | 'row'): { value: string; count: number }[]`:
- Reads `this._lastCells` and extracts unique values for the given axis field
- For each unique value, sums the `.count` property of matching cells
- Returns sorted alphabetically by value
- Uses `String(cell[axisField] ?? (dimension === 'row' ? 'None' : 'unknown'))` to match existing null handling

Add private method `_openFilterDropdown(anchorEl: HTMLElement, axisField: string, dimension: 'col' | 'row'): void`:
- Calls `this._closeFilterDropdown()` first (single dropdown at a time)
- Computes position relative to _rootEl using getBoundingClientRect delta + scrollTop/scrollLeft offset
- Creates `<div class="sg-filter-dropdown">` with position:absolute, z-index 20, background/border/border-radius/padding/min-width 180px/max-height 280px/overflow-y auto/box-shadow
- Check if dropdown would extend beyond _rootEl bottom; if so, position above anchor instead of below
- Builds checkbox list: for each value from `_getAxisValues()`, create `<label>` with `<input type="checkbox">` + `value (count)` text
- If filter is active (hasAxisFilter), check only the values in getAxisFilter; else check all
- Each checkbox `change` event: collect all checked values into array, call `this._filter.setAxisFilter(axisField, checkedValues)`. If no values checked, setAxisFilter with [] (which clears the filter per FILT-05 semantics from Plan 01)
- Append dropdown to `this._rootEl` (NOT _gridEl — must survive _renderCells clearing)
- Store in `this._filterDropdownEl`
- Set up click-outside dismiss via `requestAnimationFrame(() => { document.addEventListener('click', ..., { capture: true }) })`
- Set up Escape key dismiss via `document.addEventListener('keydown', ...)` checking `e.key === 'Escape'`

Add private method `_closeFilterDropdown(): void`:
- If _filterDropdownEl exists, remove from DOM
- Remove document click listener (capture phase)
- Remove document keydown listener
- Set _filterDropdownEl to null

Wire into `_renderCells()`: After creating each leaf column header cell (where sort icon is added), also call `_createFilterIcon(colField, 'col')` and append to the header cell. Same for row headers with `_createFilterIcon(rowField, 'row')`.
