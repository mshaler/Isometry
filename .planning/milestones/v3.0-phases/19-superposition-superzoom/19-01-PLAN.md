    **Part A: Create `src/views/supergrid/SuperZoom.ts`**

    A self-contained module that handles zoom interaction and CSS Custom Property updates.

    **Exported pure functions (testable without DOM):**
    - `normalizeWheelDelta(e: WheelEvent): number` — normalize deltaY by deltaMode, cap at +-24
    - `wheelDeltaToScaleFactor(dy: number): number` — asymmetric scale formula from RESEARCH.md:
      ```
      dy <= 0 ? 1 - (WHEEL_SCALE_SPEEDUP * dy) / 100
              : 1 / (1 + (WHEEL_SCALE_SPEEDUP * dy) / 100)
      ```

    **Constants:**
    - `BASE_COL_WIDTH = 120` (px at 1x zoom)
    - `BASE_ROW_HEIGHT = 40` (px at 1x zoom)
    - `WHEEL_SCALE_SPEEDUP = 2.0`

    **SuperZoom class:**
    ```typescript
    export class SuperZoom {
      private _rootEl: HTMLElement | null = null;
      private _gridEl: HTMLElement | null = null;
      private _positionProvider: SuperPositionProvider;
      private _onZoomChange: ((zoomLevel: number) => void) | null;
      private _wheelHandler: ((e: WheelEvent) => void) | null = null;
      private _keyHandler: ((e: KeyboardEvent) => void) | null = null;

      constructor(positionProvider: SuperPositionProvider, onZoomChange?: (zoomLevel: number) => void);

      attach(rootEl: HTMLElement, gridEl: HTMLElement): void;
      detach(): void;
      applyZoom(): void;  // reads positionProvider.zoomLevel, updates CSS Custom Properties
      resetZoom(): void;  // sets positionProvider.zoomLevel = 1.0, calls applyZoom()
    }
    ```

    **attach(rootEl, gridEl):**
    - Stores refs to rootEl and gridEl
    - Adds non-passive wheel listener on rootEl (`{ passive: false }`)
    - Wheel handler: if `!e.ctrlKey` return (let scroll happen). If `e.ctrlKey`, `e.preventDefault()`, compute scale factor, update `positionProvider.zoomLevel`, call `applyZoom()`, call `_onZoomChange` callback if provided.
    - Adds keydown listener on `document` for Cmd+0 reset (metaKey && key === '0')

    **applyZoom():**
    - Reads `this._positionProvider.zoomLevel`
    - Computes: `colWidth = Math.round(BASE_COL_WIDTH * zoomLevel)`, `rowHeight = Math.round(BASE_ROW_HEIGHT * zoomLevel)`
    - Sets on gridEl: `--sg-col-width: ${colWidth}px`, `--sg-row-height: ${rowHeight}px`, `--sg-zoom: ${zoomLevel}`
    - The `--sg-zoom` property is used by grid cells for `font-size: calc(12px * var(--sg-zoom))`, `padding: calc(4px * var(--sg-zoom))` etc. per user decision ("single --sg-zoom drives all scaling")

    **detach():**
    - Removes wheel listener from rootEl
    - Removes keydown listener from document
    - Nulls refs
