## Technical Decisions

### Dual-Layer Selection Architecture (SEL-SYNC-01)

Maintain both local selection state (SuperGridCSS context) and global selection state (SelectionContext):
- **Local state**: Handles visual feedback, path-based selection (rowPath, colPath)
- **Global state**: Handles cross-component coordination, ID-based selection (node IDs)

### Feature Flag Pattern (FEATURE-FLAG-01)

`USE_CSS_GRID_SUPERGRID` constant controls renderer selection:
- `true`: Use SuperGridCSS (React + CSS Grid)
- `false`: Use D3.js SuperGrid (default)

### Reactive Theme Propagation (THEME-SYNC-01)

Rely on React's built-in re-rendering for theme propagation, no custom subscription needed:
- ThemeContext changes trigger automatic re-renders
- `isNeXTSTEP` computed value passes to SuperGridCSS `theme` prop
- No manual useEffect or subscription code
