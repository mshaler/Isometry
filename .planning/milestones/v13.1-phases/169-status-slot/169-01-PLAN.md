    **2. Create src/superwidget/statusSlot.ts:**

    Export three functions:

    `formatRelativeTime(isoString: string): string` — pure function:
    - Compute `diffMs = Date.now() - new Date(isoString).getTime()`
    - `diffMs < 60_000` -> `'just now'`
    - `diffMs < 3_600_000` -> `'{Math.floor(diffMs / 60_000)} min ago'`
    - `diffMs < 86_400_000`:
      - `hours === 1` -> `'1 hour ago'`
      - else -> `'{hours} hours ago'`
    - `diffMs < 172_800_000` -> `'yesterday'`
    - else -> short date via `new Date(isoString).toLocaleDateString('en-US', { month: 'short', day: 'numeric' })` (e.g. `'Apr 19'`)

    `renderStatusSlot(statusEl: HTMLElement): void` — one-time DOM setup:
    - Create `div.sw-status-bar` wrapper
    - Create 3 `span.sw-status-bar__item` with `data-stat` attributes: `cards`, `connections`, `last-import`
    - Create 2 `span.sw-status-bar__sep` with textContent `'\u00B7'` (middle dot)
    - Set initial text: `'0 cards'`, `'0 connections'`, `'No imports yet'` (per D-03)
    - Append wrapper to statusEl
    - Do NOT touch statusEl's `data-render-count` attribute

    `updateStatusSlot(statusEl: HTMLElement, stats: { card_count: number; connection_count: number; last_import_at: string | null }): void`:
    - Query spans via `statusEl.querySelector('[data-stat="cards"]')`, etc.
    - If any span is null, return silently (silent failure per UI-SPEC)
    - Set cards span textContent: `stats.card_count === 1 ? '1 card' : '{N} cards'`
    - Set connections span textContent: `stats.connection_count === 1 ? '1 connection' : '{N} connections'`
    - Set last-import span textContent: `stats.last_import_at ? 'Imported ' + formatRelativeTime(stats.last_import_at) : 'No imports yet'`
    - Do NOT touch statusEl's `data-render-count` — this is the STAT-04 guarantee
