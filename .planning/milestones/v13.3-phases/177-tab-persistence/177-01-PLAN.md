Throws Error on invalid shape. Sets internal _tabs and _activeTabSlotId. Does NOT notify (same as PAFVProvider pattern — snap to state on restore).
    - `resetToDefaults(): void` — creates single tab via `makeTabSlot()`, sets _activeTabSlotId to that tab's tabId. Does NOT notify.
    - `subscribe(callback: () => void): () => void` — adds to _subscribers, returns function that removes it
    - `setTabs(tabs: TabSlot[], activeTabSlotId: string): void` — sets _tabs and _activeTabSlotId, calls _scheduleNotify()
    - `getTabs(): readonly TabSlot[]` — returns _tabs
    - `getActiveTabSlotId(): string` — returns _activeTabSlotId
    - `_scheduleNotify(): void` (private) — microtask batching via queueMicrotask with _pendingNotify guard (identical to PAFVProvider pattern)

    Type guard function (module-level, not exported):
    - `isTabStateShape(value: unknown): boolean` — checks for object with `tabs` (Array, length > 0, each element has tabId string + label string + projection object) and `activeTabSlotId` (string)

    Add a new public async method to StateManager class:
    ```typescript
    async restoreKey(key: string): Promise<void>
    ```
    Implementation:
    1. Get provider from `_providers.get(key)` — return if undefined
    2. Compute storage key via `_storageKey(key)`

    5. Try: parse JSON, run `_migrateState(key, parsed)`, call `provider.setState(migrated)`
    6. Catch: log warning `[StateManager] Failed to restore provider "${key}": ${message}. Resetting to defaults.`, call `provider.resetToDefaults()`
