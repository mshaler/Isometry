  <behavior>
    - GalleryView renders cards as uniform tiles in CSS Grid
    - Resource cards: img tag with body_text as src; onerror replaces with icon
    - Non-resource cards: large CARD_TYPE_ICONS character centered, name below
    - Responsive column count based on container.clientWidth / TILE_WIDTH
    - Tile dimensions: 240px wide, 160px tall (larger than GridView's 180x120, per CONTEXT.md)
  </behavior>
  <action>
    1. Create `src/views/GalleryView.ts`:
       - Import type { IView, CardDatum } from './types'
       - Import { CARD_TYPE_ICONS } from './CardRenderer'

       Constants:
       ```typescript
       const GALLERY_TILE_WIDTH = 240;
       const GALLERY_TILE_HEIGHT = 160;
       ```

       GalleryView class implements IView:

       Private fields: container, grid (HTMLDivElement)

       mount(container):
       - Store container reference
       - Create div.gallery-grid
       - Set grid styles: display=grid, gap=16px
       - Append grid to container

       render(cards):
       - Read container.clientWidth
       - Compute cols = Math.max(1, Math.floor(clientWidth / GALLERY_TILE_WIDTH))
       - Set grid.style.gridTemplateColumns = `repeat(${cols}, ${GALLERY_TILE_WIDTH}px)`
       - Clear existing tiles from grid
       - For each card, create a tile:

       renderGalleryTile(d: CardDatum):
       - Create div.gallery-tile
       - Set data-id = d.id
       - Set tile dimensions: width=GALLERY_TILE_WIDTH, height=GALLERY_TILE_HEIGHT
       - If d.card_type === 'resource' && d.body_text:
         - Create img.tile-image with src=d.body_text, alt=d.name
         - Set img dimensions to fill tile content area
         - img.onerror = () => { img.replaceWith(makeFallbackIcon(d)) }
         - Append img to tile
       - Else:
         - Append makeFallbackIcon(d) to tile
       - Create span.tile-name with d.name
       - Append name to tile
       - Return tile

       makeFallbackIcon(d: CardDatum):
       - Create div.tile-icon
       - Set textContent = CARD_TYPE_ICONS[d.card_type] ?? 'N'
       - Style: large font size, centered, fills content area
       - Return icon div

       destroy():
       - Remove grid from container
       - Null container and grid references

    2. Run tests — all GalleryView tests must pass (GREEN)
    3. Run full view test suite to verify no regressions
    4. REFACTOR if needed: extract tile creation helpers
  </action>
