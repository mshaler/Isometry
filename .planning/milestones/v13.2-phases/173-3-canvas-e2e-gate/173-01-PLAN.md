**Test 1 — INTG-01: 6-direction transition matrix**
Single test with 6 sequential transitions. Each uses `commitAndAssert` then verifies the previous canvas type is gone (`.toHaveCount(0)`).

Transition sequence:
1. Explorer -> View: commit viewBoundProjection, assert View visible, Explorer gone
2. View -> Explorer: commit explorerProjection, assert Explorer visible, View gone
3. Explorer -> Editor: commit editorProjection, assert Editor visible, Explorer gone
4. Editor -> Explorer: commit explorerProjection, assert Explorer visible, Editor gone
5. Editor -> View: start from Editor (commit editorProjection first), then commit viewBoundProjection, assert View visible, Editor gone
6. View -> Editor: commit editorProjection, assert Editor visible, View gone

Each step: call `commitAndAssert`, then assert previous `[data-canvas-type="X"]` has count 0.

**Test 2 — INTG-02: 9-view cycle DOM leak detection (per D-05, D-06)**
Start by committing viewBoundProjection. Then loop through ALL_VIEWS:
```typescript
for (const view of ALL_VIEWS) {
  await page.evaluate((v) => {
    const sw = (window as any).__sw;
    // ViewCanvas uses Projection.activeTabId to encode view type
    sw.commitProjection({
      canvasType: 'View',
      canvasBinding: 'Bound',
      zoneRole: 'secondary',
      canvasId: 'view-1',
      activeTabId: v,
      enabledTabIds: [v],
    });
  }, view);
  // Wait for view to render — canvas slot must have exactly 1 child
  await page.waitForTimeout(view === 'network' ? 2000 : 500);
  await expect(page.locator('[data-slot="canvas"] > *')).toHaveCount(1);
  await expect(page.locator('[data-canvas-type="View"]')).toBeVisible();
}
```

**Test 3 — INTG-04: Rapid switching stress test (per D-07, D-08)**
Fire 3 commitProjection calls synchronously in one page.evaluate block with no awaits:
```typescript
await page.evaluate((projs) => {
  const sw = (window as any).__sw;
  sw.commitProjection(projs[0]); // Explorer
  sw.commitProjection(projs[1]); // View
```

- No console errors: attach `page.on('console', ...)` listener before the burst, collect error-level messages, assert array is empty after settling
