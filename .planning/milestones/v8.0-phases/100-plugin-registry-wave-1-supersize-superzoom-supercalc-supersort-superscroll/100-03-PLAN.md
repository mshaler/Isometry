    **SuperCalcFooter.ts** (~120 LOC):
    - Export type: `AggFunction = 'SUM' | 'AVG' | 'COUNT' | 'MIN' | 'MAX' | 'NONE'`
    - Export pure function:
      ```typescript
      export function computeAggregate(fn: AggFunction, values: (number | null)[]): number | null
      ```
      - SUM: reduce, treating null as 0. Return null if values is empty.
      - AVG: sum of non-nulls / count of non-nulls. Return null if no non-nulls.
      - COUNT: count of non-null values
      - MIN: Math.min of non-nulls. Return null if no non-nulls.
      - MAX: Math.max of non-nulls. Return null if no non-nulls.
      - NONE: return null

    - Internal shared state: `aggFunctions: Map<number, AggFunction>` (colIdx -> function). Default: SUM for all.

    Factory `createSuperCalcFooterPlugin(sharedConfig?: { aggFunctions: Map<number, AggFunction> })`:
    - If sharedConfig provided, use it (allows config plugin to set functions). Otherwise use internal Map.

    - `afterRender(root, ctx)`:
      - Cast ctx to RenderContextWithLayout to read layout sizing
      - Find or create `.pv-calc-footer` container as a sibling of the scroll container (sticky at bottom)
      - Container style: `position: sticky; bottom: 0; z-index: 15; display: flex;`
      - Container background: `var(--pv-accent-light)`
      - For each visible column (0 to visibleCols.length - 1):
        - Collect all cell values from ctx.data for this column
        - Get aggFunction from `sharedConfig.aggFunctions.get(colIdx) ?? 'SUM'`
        - Compute aggregate via `computeAggregate(fn, columnValues)`
        - Create cell div with class `pv-calc-cell`:
          - Width matches column width: `layout.colWidths.get(colIdx) ?? layout.cellWidth` px
          - If colIdx === 0: add `border-left: 3px solid var(--pv-accent)`
          - Content: if result !== null → `<span class="pv-calc-glyph">∑</span> ${result}` (glyph changes based on fn: ∑ for SUM, x̄ for AVG, # for COUNT, ↓ for MIN, ↑ for MAX)
          - If result === null → `—` in `var(--pv-cell-empty-fg)` color
          - Font: 13px monospace `var(--pv-number-font)`, weight 600, right-aligned
      - First cell gets `border-left: 3px solid var(--pv-accent)` per UI-SPEC

    - `destroy()`: Remove footer container
