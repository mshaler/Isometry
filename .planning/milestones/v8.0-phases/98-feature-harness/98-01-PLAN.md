# Plan 98-01: Plugin Registry + Feature Harness Shell

## Goal
Build a composable plugin registry for SuperGrid features with a visual test harness
that provides sub-feature-level toggles. Each Super* capability is a plugin that hooks
into the render pipeline. The harness lets developers enable/disable individual features
and sub-features to test in isolation and composition.

## Architecture

### Plugin Interface

```typescript
interface PluginHook {
  /** Transform cell data before D3 join (e.g., virtual scrolling filters rows). */
  transformData?(cells: CellPlacement[], ctx: RenderContext): CellPlacement[];
  /** Modify grid sizing/layout before render (e.g., zoom scales cell dimensions). */
  transformLayout?(layout: GridLayout, ctx: RenderContext): GridLayout;
  /** Inject additional DOM after main render (e.g., calc footer, selection overlay). */
  afterRender?(root: HTMLElement, ctx: RenderContext): void;
  /** Handle pointer events (return true to consume). */
  onPointerEvent?(type: string, e: PointerEvent, ctx: RenderContext): boolean;
  /** Handle scroll events (e.g., virtual scrolling recalculates visible range). */
  onScroll?(scrollLeft: number, scrollTop: number, ctx: RenderContext): void;
  /** Cleanup on disable or destroy. */
  destroy?(): void;
}
```

### Plugin Metadata (Sub-Feature Taxonomy)

```typescript
interface PluginMeta {
  id: string;           // e.g., 'superstack.collapse'
  name: string;         // e.g., 'Collapse Groups'
  category: string;     // e.g., 'SuperStack'
  description: string;
  dependencies: string[]; // other plugin IDs required
  defaultEnabled: boolean;
}
```

### Registry

```typescript
class PluginRegistry {
  register(meta: PluginMeta, factory: () => PluginHook): void;
  enable(id: string): void;   // auto-enables dependencies
  disable(id: string): void;  // auto-disables dependents
  isEnabled(id: string): boolean;
  getEnabled(): PluginMeta[];
  getAll(): PluginMeta[];
  getByCategory(): Map<string, PluginMeta[]>;

  // Pipeline execution
  runTransformData(cells, ctx): CellPlacement[];
  runTransformLayout(layout, ctx): GridLayout;
  runAfterRender(root, ctx): void;
}
```

### Data Source Progression

```typescript
type DataSource = 'mock' | 'alto-json' | 'sqlite';
```

The harness provides a data source selector. Each source implements a common interface:
```typescript
interface DataProvider {
  name: string;
  load(): Promise<Map<string, number | null>>;
  getDimensions(): HeaderDimension[];
}
```

## Feature Taxonomy (Full Catalog)

### Base (always on)
- `base.grid` — D3 data join cell rendering
- `base.headers` — Two-layer header spanning
- `base.config` — Config panel with DnD axis assignment

### SuperStack
- `superstack.spanning` — Multi-level run-length header spans
- `superstack.collapse` — Click header to collapse group (depends: spanning)
- `superstack.aggregate` — Aggregate values on collapse (depends: collapse)

### SuperZoom
- `superzoom.slider` — Zoom slider UI control
- `superzoom.scale` — CSS transform cell scaling (depends: slider)

### SuperSize
- `supersize.col-resize` — Per-column drag-to-resize handles
- `supersize.header-resize` — Header width/height resize (corner handles)
- `supersize.uniform-resize` — Uniform cell resize (bottom-right handle)

### SuperDensity
- `superdensity.mode-switch` — 1x/2x/5x toggle UI
- `superdensity.mini-cards` — 5x mode mini-card rendering (depends: mode-switch)
- `superdensity.count-badge` — 1x mode count-only rendering (depends: mode-switch)

### SuperCalc
- `supercalc.footer` — Sticky footer row with aggregate values
- `supercalc.config` — Per-column aggregate function selector (depends: footer)

### SuperScroll
- `superscroll.virtual` — Virtual scrolling for 10K+ rows
- `superscroll.sticky-headers` — Sticky row/col headers during scroll

### SuperSearch
- `supersearch.input` — Search input UI
- `supersearch.highlight` — Cell text highlighting (depends: input)

### SuperSelect
- `superselect.click` — Click-to-select cell
- `superselect.lasso` — SVG lasso multi-select (depends: click)
- `superselect.keyboard` — Shift/Cmd keyboard modifiers (depends: click)

### SuperAudit
- `superaudit.overlay` — Change tracking visual overlay
- `superaudit.source` — Source provenance badges (depends: overlay)

### SuperSort
- `supersort.header-click` — Click header to sort
- `supersort.chain` — Multi-column sort chain (depends: header-click)
