<objective>
Fix SuperGrid axis grip DnD so same-dimension reorder and cross-dimension transpose actually commit on pointer release. Also fix pointer-event occlusion on stacked/nested column header grips.

Purpose: UAT tests 1-3 and 6 all fail because _handlePointerDrop requires the pointer to be geometrically inside a 6px drop zone at pointerup time. Same-dimension reorder never hits a drop zone (the user drags within the header area, not over a 6px edge strip). Cross-dimension transpose is nearly impossible because 6px is too small to reliably hit. Additionally, UAT test 2 reports that stacked/nested column headers cannot be moved — the drop zones (z-index:10) sit above header grips (z-index:2), potentially occluding pointer events on grips in the first header row. User also reported "alignment issues" in the column header area.

Output: Working SuperGrid axis grip reorder and transpose in WKWebView
</objective>


**Root cause:** The axis drop zones are created in mount() at lines 644-671 with `pointer-events: auto` and `z-index: 10`. Column headers have `z-index: 2` and are sticky at `top: 0`. The col drop zone (also at `top: 0`, `z-index: 10`) sits ABOVE the first row of column header grips, intercepting pointer events. This is worse when Task 1 enlarges drop zones to 40px during drag — but the problem exists even at 6px for the topmost header row.

**Fix: Toggle drop zone pointer-events based on drag state.**

1. In mount() where drop zones are created (lines 644-671), change BOTH drop zones from `pointer-events: auto` to `pointer-events: none`:
   ```typescript
   colDropZone.style.pointerEvents = 'none';  // was 'auto' — enable only during drag
   rowDropZone.style.pointerEvents = 'none';  // was 'auto' — enable only during drag
   ```

2. In BOTH grip `pointerdown` handlers (lines ~4260 and ~4488), alongside the drop zone enlargement code from Task 1, also enable pointer-events:
   ```typescript
   if (this._rootEl) {
       const colDZ = this._rootEl.querySelector<HTMLElement>('.axis-drop-zone--col');
       const rowDZ = this._rootEl.querySelector<HTMLElement>('.axis-drop-zone--row');
       if (colDZ) { colDZ.style.height = '40px'; colDZ.style.pointerEvents = 'auto'; }
       if (rowDZ) { rowDZ.style.width = '40px'; rowDZ.style.pointerEvents = 'auto'; }
   ```
