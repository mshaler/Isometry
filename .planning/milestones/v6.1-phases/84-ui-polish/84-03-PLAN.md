# Plan 84-03: Replace alert/confirm with in-app AppDialog

## Objective

Three call sites use native browser blocking dialogs that break the visual language and block the JS thread. Create a lightweight non-blocking in-app dialog primitive using native `<dialog>` element and migrate all three call sites.

<tasks>

<task id="1">
**Create src/ui/AppDialog.ts**

Lightweight imperative dialog with single public API:

```typescript
export interface AppDialogOptions {
  title: string;
  message: string;
  variant: 'info' | 'confirm';
  confirmLabel?: string;   // default: 'Confirm'
  cancelLabel?: string;    // default: 'Cancel'
}

export const AppDialog = {
  show(options: AppDialogOptions): Promise<boolean> { /* ... */ }
};
```

Implementation requirements:
- Native `<dialog>` element (accessible by default)
- Append to document.body; remove self on resolution
- Focus trap: Tab/Shift-Tab cycle between interactive elements only
- Escape → resolves false
- Backdrop click → resolves false
- Apply CSS variables from design-tokens.css for theming
</task>

<task id="2">
**Create src/styles/app-dialog.css**

Minimal styles using existing CSS variables:
- `.app-dialog-backdrop` — overlay
- `.app-dialog` — panel
- `.app-dialog__title` — heading
- `.app-dialog__message` — body text
- `.app-dialog__actions` — button row
- `.app-dialog__btn` — shared button base
- `.app-dialog__btn--confirm` — primary action
- `.app-dialog__btn--cancel` — secondary action

Import this CSS in AppDialog.ts.
</task>

<task id="4">
**Migrate PropertiesExplorer.ts:468 (_handleResetAll)**

Convert to async:
```typescript
private async _handleResetAll(): Promise<void> {
  const count = this._config.schema?.getOverrides().size ?? 0;
  if (count === 0) return;
  const confirmed = await AppDialog.show({
    variant: 'confirm',
    title: 'Reset Mappings',
    message: `Reset ${count} custom mapping${count > 1 ? 's' : ''} to defaults?`,
    confirmLabel: 'Reset',
  });
  if (!confirmed) return;
  // ... existing reset logic
}
```

Update button event listener to `() => { void this._handleResetAll(); }`.
</task>
