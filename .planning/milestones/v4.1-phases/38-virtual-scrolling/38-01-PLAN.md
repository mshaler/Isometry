<objective>
Build the SuperGridVirtualizer module with TDD, add CSS content-visibility progressive enhancement, and integrate both into SuperGrid for row-level windowing at 10K+ card scale.

Purpose: SuperGrid currently renders ALL rows to DOM, causing jank at scale. This plan adds a two-layer virtualization strategy: CSS content-visibility (browser-native paint skipping) and custom row windowing (only visible + overscan rows get DOM nodes).

Output: Working virtualizer integrated into SuperGrid -- grids with >100 leaf rows render only visible rows; grids with <=100 rows get content-visibility only. Frozen headers preserved. Scrollbar accurate.
</objective>
