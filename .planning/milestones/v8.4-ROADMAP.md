## Overview

Removes the ViewZipper horizontal tab strip and consolidates all view-switching into SidebarNav Visualization Explorer. Adds auto-cycle Play/Stop button and unifies crossfade transitions across all three view-switch paths.

## Phases

### Phase 108: Consolidate View Navigation
**Goal**: Remove ViewZipper, make SidebarNav the sole view navigation UI with crossfade and auto-cycle
**Depends on**: None (standalone cleanup)
**Plans**: 2 plans

Plans:
- [x] 108-01-PLAN.md — Remove ViewZipper and consolidate into SidebarNav
- [x] 108-02-PLAN.md — Add auto-cycle Play/Stop to Visualization Explorer

**Details:**
- Deleted ViewZipper.ts and view-zipper.css
- SidebarNav Visualization Explorer defaults to expanded (data-state=visible)
- .view-crossfade CSS class replaces .vzip-transition-frame in workbench.css
