# Plan 34-01: Reminders Adapter (EventKit)

<objective>
Create RemindersAdapter.swift that uses EventKit to fetch incomplete + last-30-days completed reminders from macOS Reminders, maps each EKReminder to a CanonicalCard with card_type "task", and yields them through the existing NativeImportAdapter AsyncStream protocol.
</objective>

<tasks>

## Task 1: Create RemindersAdapter.swift

Create `native/Isometry/Isometry/RemindersAdapter.swift`:

- Conforms to `NativeImportAdapter` protocol
- `sourceType = "native_reminders"`
- Permission: uses `EKEventStore.authorizationStatus(for: .reminder)` and `requestFullAccessToReminders()` (iOS 17+ / macOS 14+)
- Handles all auth states: `.fullAccess`, `.authorized` (legacy), `.denied`, `.notDetermined`, `.restricted`, `.writeOnly`
- `fetchCards()`: fetches incomplete reminders + completed in last 30 days via `predicateForIncompleteReminders` and `predicateForCompletedReminders`
- Maps each `EKReminder` to `CanonicalCard`:
  - `card_type: "task"` (RMDR-05)
  - `source_id: calendarItemIdentifier` (RMDR-03 dedup key)
  - `folder: calendar.title` (RMDR-02 list name)
  - `due_at`: from `dueDateComponents`
  - `completed_at`: from `completionDate` if `isCompleted`
  - `status`: "done" or "active"
  - `priority`: map EventKit 0/1/5/9 → 0/3/2/1
  - `content`: notes text + recurrence description (RMDR-04)
  - Location from alarm `structuredLocation` if present
  - All dates via ISO8601DateFormatter (not CoreData — EventKit returns Foundation Date)

**Commit**: `feat(34-01): add RemindersAdapter for EventKit-based reminder import`

</tasks>
