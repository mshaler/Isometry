# Plan 35-01: Notes Adapter (Direct SQLite3)

<objective>
Create NotesAdapter.swift that reads NoteStore.sqlite directly via the C SQLite3 API (no dependencies), extracts note titles, folders, dates, and ZSNIPPET preview content, detects schema version at runtime, skips encrypted notes with a count report, and yields CanonicalCards with card_type "note".
</objective>

<tasks>

## Task 1: Create NotesAdapter.swift

Create `native/Isometry/Isometry/NotesAdapter.swift`:

- Conforms to `NativeImportAdapter` protocol
- `sourceType = "native_notes"`
- **No external dependencies** — uses `import SQLite3` (system libsqlite3)
- Permission: checks file access to `~/Library/Group Containers/group.com.apple.notes/NoteStore.sqlite`
  - If accessible → `.granted`
  - If not → `.denied` (requires Full Disk Access in System Settings)
  - `requestPermission()` opens System Settings to Full Disk Access pane via PermissionManager
- Database access strategy:
  - Uses PermissionManager.copyDatabaseToTemp() for safe copy-then-read (FNDX-04)
  - Opens copy with `sqlite3_open_v2(path, &db, SQLITE_OPEN_READONLY, nil)`
  - Cleans up temp copy after fetch completes
- NOTE-06: Schema version detection via `PRAGMA table_info(ZICCLOUDSYNCINGOBJECT)`:
  - Check column names → determine ZTITLE1 vs ZTITLE2, ZCREATIONDATE3 vs ZCREATIONDATE, ZACCOUNT3 vs ZACCOUNT4
  - Build query dynamically based on detected columns
- Query: SELECT notes from ZICCLOUDSYNCINGOBJECT where not deleted and not a folder/account:
  - Filter by Z_ENT entity type or by presence of ZIDENTIFIER + title column
  - NOTE-04: Detect encrypted notes (ZCRYPTOINITIALIZATIONVECTOR IS NOT NULL or ZISPASSWORDPROTECTED = 1) — skip and count
- Maps each row to CanonicalCard:
  - `card_type: "note"`
  - `source_id: ZIDENTIFIER` (NOTE-05 dedup key)
  - `name`: from title column
  - `content`: ZSNIPPET (100-char preview, NOTE-01)
  - `created_at / modified_at`: CoreDataTimestampConverter (dates are CoreData epoch)
  - NOTE-02: Folder hierarchy via self-join on ZFOLDER → parent Z_PK, build path string
  - NOTE-03: Extract #hashtag patterns from ZSNIPPET → tags array
- Yields cards in batches of 500 (natural batch from single query iteration)

**Commit**: `feat(35-01): add NotesAdapter for direct NoteStore.sqlite import`
